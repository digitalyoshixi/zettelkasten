# C++ Build Process
1. Turn C++ files into .obj files
2. Link all .obj files and [[Library Files|.lib]] files to create a .exe file
### Release Vs Debug
When compiling a debug version, it is more bloated because debug information used to assist for debugging is also compiled.
Release versions are always faster.
