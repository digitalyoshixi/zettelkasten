Used to track system resource usage.
- Application usage, energy usage, push notifications, network connectivity, data for 30 days
Enabled by default and can be viewed in [[Task manager]].
# SRUM Database
```
C:\Windows\System32\sru
```
Can use [[SrumECmd]] to analyze.
