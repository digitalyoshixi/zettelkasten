These are built-in `..` statements that return a list based off range start and end values.
# Range Iterator
### Start,End,Step
```rust
(2..) // starting from 2
(..2) // ending at 2
(0..3) // start at 0, end at 3
```
##### Natural Numbers
```rust
let natural_numbers = 1..;
```
