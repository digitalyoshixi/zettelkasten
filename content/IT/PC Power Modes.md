# Mechanical Off
All components are powered off except for the [[Real Time Clock|RTC]]
# Soft Power-Off / Sleep
Only components on are:
- [[Keyboard]]
- [[Local Area Network|LAN]]
- [[Universal Serial Bus|USB]]
in order to wake the system from input or [[Network Interface Controller|Wake-on-LAN]]
# Suspend / Hibernate
Same as sleep, but RAM is temporarily stored on the disk and wakeup takes longer as a result of the RAM rewriting process.
# Soft Reboot
Term for **PCs**.
Can be triggered inside the operating system or with a dedicated button.
Does not power the entire system off. Caused by windows Fast Startup, which skips the [[System Setup Utility|BIOS Menu]]
# Soft Reset
Term for **mobile devices**.
Can be triggered inside the operating system or with a dedicated button or by power cycling the device.
If the device wont shut down, then remove the battery.
# Hard Reboot
Term for **PCs**.
Holding the [[Soft Power Button]] down for 10s, and then turning the device on again. Cuts off all power to the system for a brief while. Not recommended
# Hard Reset (Factory Reset)
Reinstalling the [[IT/Operating System|OS]] and removing all user data.
