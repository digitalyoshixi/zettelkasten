# Nothing Shown
Flip through video sources
# Image too bright/dark
- In the monitor, use the buttons to change the image-adjustment settings
- Replace backlight
- Replace [[Inverter]]
- OS [[redshift]] or nightlight software
# Flashing Screen
- Check physical cable connectivity
- Change to shorter cable
# Bad Pixel
Caused by manufacturing defects
- Dead pixels are off no matter what
- Lit pixels are always white
- Stuck pixels are pixels at a specific color that wont change
If you have a certain number of bad pixels, then this is under warranty. See if you can get a replacement
# Image Persistence / Burn-ins
![[Graphics Troubleshooting-20240731200401585.webp|571]]
You see afterimages.
- These go away after power cycling
- For LCD monitors, display a white image for an extended period
Sometimes they may be display burn-ins and you must replace the screen
OLED monitors have options to reduce and minimize burn-ins.
# Small Icons
Higher resolution monitors make icons smaller.
Reducing resolution makes things look fuzzier, so instead just use software to manually increase the size of icons.
# Cleaning Monitors
Use dry micro-fiber cloths
# Bad Drivers
Boot into Windows Safe mode to see if drivers are the issue
# VGA Mode
- Press F8 
- A very basic resolution compatible with most monitors
# Pink/Purple Discolorations
- Faulty cable
- Loose cable
