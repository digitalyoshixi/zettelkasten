### Dynamic RAM (DRAM)
![[Random Access Memory-20240523005301778.webp]]
- Data stored when plugged in is held only temporary and expires after a few seconds. It [[Memory Refresh|Refreshes]] itself to preserve the same data
- This refresh can cause some delays for older CPU's which have not implemented latency adjustments.
