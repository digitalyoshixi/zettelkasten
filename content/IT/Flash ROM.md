Flash ROM is a type of ROM that allows certain rewriting.
# BIOS Flashing
ROM chips may occasionally need to update to support new technology. This is called ROM flashing or firmware update. 
You can run a program on your terminal to run a firmware update to flash the ROM.
# Process
1. Insert a USB drive containing updated BIOS file
2. Use firmware updating utility in [[System Setup Utility|BIOS Menu]]
