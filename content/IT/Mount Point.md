Turning a partition into a folder on another drive.
![[Mount Point-20240704041937208.webp|399]]
This essentially routes all files going into one folder to be placed into the other drive.
In the [[Windows Disk Management]] software, you can mount by making new partition and assigning it to be treated like a folder.
