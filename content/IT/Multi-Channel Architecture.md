Ability for multiple RAM sticks which have the:
- Same [[Double Data Rate|DDR SDRAM]] type
- Same speed
- Same capacity
to work as one stick
# Dual-Channel
[[Random Access Memory|DDR SDRAM]] sticks will often have dual-channel architecture support which means 2 sticks of DDR can be used as the same RAM channel.
![[Dual-Channel Architecture-20240523182756517.webp]]
Dual-channel architecture allows for increased throughput overall.
Has a 128-bit data bus
# Triple-Channel
![[Triple-Channel Architecture-20240726220849709.webp]]
Ability for 3 sticks of RAM to be used as the same channel.
Has a 192-bit bus if all 3 are used
# Quad-Channel
![[Quad-Channel Architecture-20240726221151799.webp]]
Ability for 4 sticks of RAM to occupy the same data stream.
Allows for 4x data rate
