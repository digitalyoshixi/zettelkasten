Supports volumes up to 1 exabyte and is backwards compatible with [[ext2]] and [[ext3]].
- Can support up to 64 [[Binary|Zetabytes]] max volume size
- Can support single files up to 16 [[Binary|Terrabytes]]
- Can support volumes up to 1 [[Binary|Yottabyte]]
- Can have [[Cluster]] up to 64 [[Binary|Kibibyte]]
