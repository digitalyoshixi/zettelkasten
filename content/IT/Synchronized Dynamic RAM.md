### Synchronized Dynamic RAM (SDRAM)
![[Random Access Memory-20240523012416099.webp|259]]
- You can install multiple sticks into your motherboard
- Memory banks (Each separate RAM stick) split up the contiguous memory into several sections to reduce read&write times
- Synchronized with [[Crystal Oscillator|System Clock]] which allows it to send data exactly when CPU finished reading data.
- SDRAM MUST be above the System clock speed to be able to function
- Most popular type of RAM
##### Types
- [[Dual Inline Memory Module|DIMM]]
- [[Small Outline Dual In-line Memory Module|SO-DIMM]]
