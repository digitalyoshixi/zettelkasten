![[DIMM-20240523014831562.webp|439]]
One of the most common type of memory modules.
- Separate connectors on each side of the stick
- Uses 3.3 volts 
- 64-bit data width
# Installation
- All RAM must be faster or equivalent to the motherboard clock speed specifications
### Multi-Channel Installation
- Avoid choosing RAM sticks with differing [[CAS Latency]] and Clock speed
- ![[Random Access Memory-20240524155851876.webp|514]]
  Install pairs of RAM in a pair of matching colors. For example, if I had a pair of RAM, I need to install them in the yellow sockets so they match.
### How to Install
1. Ground yourself with Antistatic wrist. [[Hardware Physical Issues]]
2. Open the side panels
3. Line up the RAM and drop it in. The side panels wil automatically click in.
