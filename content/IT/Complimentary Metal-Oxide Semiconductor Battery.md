CMOS battery holds permanent data that's also writable. It holds:
- Boot password
- RAM size
- System settings
- Date/Time
The [[System Setup Utility|BIOS Menu]] allows you to modify CMOS data.
There is a giant lithium battery to ensure the CMOS is constantly powered.
