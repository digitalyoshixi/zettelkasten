# Laptop Wont Power On
- Test outlet
- Swap [[AC Adapter]]
- Remove all peripheral devices
# Poor Performance
- Reboot
- [[Task manager]] and stop heavy processes
# Battery Issues
- [[Battery Swelling]]
- Battery capacity may be limited by the OS
- [[AC Adapter]] may not be charging
- Replace battery
# Overheating
Diagnosed by:
- Graphical glitches
- Periodic reboots
- System beeps
Cool it off by:
- Setting device on a cool and hard surface
- Check if fans are working
- Clear dust
- Check battery health
# Screen Issues
- Increase brightness
- Plug into different monitor
- WIN+P to mirror displays
- Check monitor connectors
- Check rotation locks
- Update drivers if screen is flickering
# No Networking (Wifi, NFC, Bluetooth, GPS)
- Disable airplane mode
- Check antenna inside display panel
- Go closer to router
- For bluetooth, determine both devices are on and in pairing mode
- Check privacy options in [[Windows Control Panel]]
- Enable proximity support  in [[Windows Control Panel]] to enable [[Near Field Scanner|NFC]]
# Audio Issues
- Unmute device
- Plug in new speakers
- Enable then disable device in [[Windows Device Manager]]
# Input Problems
- Check keyboard connectors 
- Check NumLock
- Clean with compressed air
- Reattach laptop keys
- Update trackpad driver in [[Windows Device Manager]]
- Use touchpad diagnostics in [[System Setup Utility|BIOS Menu]]
