Machine code loaded in RAM that is running in CPU threads.
These can take the form of:
- [[Foreground Processes]]
- [[Background Process]]
- [[Child Process]]
- [[Orphan Process]]
- [[Zombie Processes]]
- [[Daemon Process]]
# Process ID (PID)
All processes will have a process ID that can be viewed with `ps -a`
