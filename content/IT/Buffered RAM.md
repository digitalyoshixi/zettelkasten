There is only 2 type of RAM.
Registered/Buffered or Unbuffered.
Registered/Buffered RAM will go to the memory banks first before being sent to [[Memory Chip Controller|MCC]] 
Unbuffered RAM will go directly to the [[Memory Chip Controller|MCC]]
# Unbuffered RAM
- Used in consumer grade hardware
# Buffered RAM
- Used in servers
- Quicker as [[Memory Chip Controller|MCC]] will only need to communicate with a few memory banks, rather than all RAM chips
