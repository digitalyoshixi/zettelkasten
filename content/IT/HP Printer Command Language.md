A set of printer commands derived from [[ASCII]].
Not a true page description language as instead of being plaintext, it uses a series of commands to represent characters.
PCL 1-5 Only prints text, and does not support graphics.
PCL 6 supports scalable fonts and line drawing.
Only supported on HP systems.
