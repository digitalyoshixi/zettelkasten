# DVD Media
looks similar to CD, but has a whole different technology to it.
The differences from CD include:
- Data storage. Can hold around 4.37-16GB of data.
- Has smaller pits than CD
- Pits are packed more densely 
- Comes in single sided (SS) and double sided (DS) formats
- Comes in single layer (SL) or dual layer (DL) reflectivity forms
### DVD-ROM
Functionally same as CD-ROM, but it can store more data.
### DVD-R
Can write once to the DVD. Functionally same as CD-R
### DVD-RW
Functionally same as CD-RW. Can rewrite a disk as many times as you want
### DVD-RW DL
Can rewrite a disk on both sides
