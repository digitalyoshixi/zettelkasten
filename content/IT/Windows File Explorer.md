# Settings
### Show Hidden Files
- Untick the *Show hidden files, folders and drives* box
- Untick the *Hide protected operating system files* box
### Show or Hide Extensions
View > Uncheck *Hide extensions for known file types*
### Disabling Indexing
- Disable indexing of system directories
