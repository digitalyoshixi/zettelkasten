### Adding Local Printer
[[Windows Control Panel]] > Devices and Printers > Add a printer or Scanner 
OR
[[Windows Settings]] > Devices > Printers & Scanners > Add a printer or Scanner 
### Adding Networked Printer
Select *The Printer I want isn't listed* > *Add a printer using TCP/IP address or hostname* > Install drivers they want you to install > Give address
OR
Add printer wizard will automatically search for drivers
