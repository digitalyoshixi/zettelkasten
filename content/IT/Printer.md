# Printer Setup
![[Printer-20240725044506710.webp|461]]
A printer is comprised of: 
- Printer software
- Printer spool of print jobs
- Printer driver
- Print device(s)
A single printer software can support several print devices, meaning you can have redundancy and productivity improvements.
