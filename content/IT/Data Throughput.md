The quantity of how much data our system can handle over a period of time.
Inverse relation to [[Data Latency]]
# Servers
Measured in Requests Per Second (RPS)
# [[Database]]
Measured in Queries Per Second (QPS)
# Data 
The rate at which an storage device can read and write data.
Expressed in Bytes per second, usually megabytes per second ($MBps$)
- [[Hard Drive|HDDs]] usually hit ~200 MBps,
- [[Solid State Drive|SSDs]] usually get 600 MBps,
- [[M.2|NVMe SSD]] usually get 2500 MBps
