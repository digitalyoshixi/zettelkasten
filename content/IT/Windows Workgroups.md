Every computer stores its users and passwords locally.
![[Windows Domains-20240704220833824.webp]]
It can be networked, but it can also be separate from the network.
There is no central authentication. All devices have the same access to every other device.
Workgroups offer no better security. They are just a way to use hardware more efficiently.
# Changing Workgroup Name
The default workgroup name is WORKGROUP
To change:
1. Settings > About PC
2. Rename this PC (Advanced)
   ![[Windows Workgroups-20240717211333687.webp]]
3. Change and then rename the workgroup to a different name
   ![[Windows Workgroups-20240717211403793.webp]]
