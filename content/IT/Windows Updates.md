Updates to the windows OS, that come as patches or new features.
# Update Types
### Quality Updates
Patches and bug fixes.
### Feature Updates
New versions of windows that include new features.
Named after the year and which half of the year it was published in.
Update 21H2 indicates that its released in second half of year 2021.
# Microsoft Patch Tuesday
Every tuesday at 10AM PST a number of security patches are released for windows
