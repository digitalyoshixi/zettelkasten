A [[Printed Circuit Board]] that an be inserted into an [[Expansion Bus|Expansion Slot]] to add functionality to a computer system.
This added functionality can be:
- [[Network Interface Card]]
- [[Graphics Processing Unit|Graphics Card]]
- [[Capture Card]]
- [[Modem]]
- [[Multi-port Ethernet]]
- etc
# Installation
1. Know if the card is compatible with your [[Operating System]]
2. Insert the card into the [[Motherboard]]
3. Provide [[Device Driver|Drivers]] for the [[Operating System]]
4. Verify the card functions
### Tips
- it is not necessary to clean your expansion card after it has been installed. Only clean it if it is before installation.
# Troubleshooting
1. Use [[Windows Device Manager]] to run diagnostics
2. Reinstall [[Device Driver]] if the device does not show in device manager. Run [[Windows Hardware Wizard]] or just reinstall the driver from website
3. Check if device is turned off in [[Complimentary Metal-Oxide Semiconductor Battery|CMOS]]
4. [[Hardware Troubleshooting]]
