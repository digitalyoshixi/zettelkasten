[[Windows Control Panel]] > Devices and Printers 
# Settings
### Layout
- **Duplex:** lets you choose single or double sided prints
- **Orientation:** lets you choose landscape or portrait
- **Multiple Page:** lets you choose how many pages you want on a single physical page
- **Scaling:** Resize the image on the page
- **Reverse:** Mirroring an image, and printing the mirror
### Paper
- **Paper Size**: Several standard sizes like A4, letter or a custom size
- **Paper Type:** thickness, coating and formats like envelope or label
- **Paper Source:** Allows for automatic paper from paper tray, or manual feed
- **Tray Settings:** if you have multiple trays, allows you to select which trays to pick from.
### Quality
- **Resolution:** dpi the image should be printed at
- Some prints allows for graphics optimization presets
- Some printers allow for toner saving options
### Other Settings
- Apply watermark
- Header/footer information
- **Collate:** order in which multiple copies are printed. if disabled, 10 copies of first page are printed, then 10 pages of second page is printed. if enabled, first copy is printed the second is printed, on and on 10 times.
