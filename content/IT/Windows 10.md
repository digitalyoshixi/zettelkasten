Windows version with the most compatibility with software.
# [[Windows 10 Installation]]
### Installation Prerequisites
- 1 GHZ CPU
- 2 GB RAM
- 32 GB Drive space
- 800x600 display
# Windows 10 Editions
![[Windows 10-20240819182304723.webp]]
All editions can be [[Windows Upgrade|In-Place Upgraded]] to higher versions or from versions of older OS models like [[Windows 8.1]]
### Windows 10 Home
Windows version that blocks a lot of great features. 
- Only supports 128GB of RAM
- Only can use [[Windows Workgroups]]
- Can't use [[Remote Desktop Protocol|RDP]] server. Can use [[Remote Desktop Protocol|RDP]] client
### Windows 10 Pro
- Support of 2TB RAM
- [[Windows Workgroups]] and [[Windows Domain]] support
- Access to [[Windows Group Policies]] through [[Windows Group Policy Editor|gpedit.msc]]
- Access to [[Windows BitLocker]]
- Access to [[Windows Remote Desktop]]
- Support for [[Assigned Access]]
### Windows 10 Pro for Workstations
- Support of 6TB RAM
- Support for [[Resilient File System]]
- All the same tools found in Win 10 Pro
### Windows 10 Enterprise
- Same as Win 10 Pro for Workstations.
- Support for [[Windows Long-Term Servicing Branch]]
- Support for [[AppLocker]]
- Support for [[BranchCache]]
### Windows 10 No license
- Cannot change desktop wallpaper
# Windows 10 CPU Architectures
### x86 (x32)
- Supports up to 4GB [[Random Access Memory|RAM]]
### x64
- Supports up to 6TB [[Random Access Memory|RAM]]
