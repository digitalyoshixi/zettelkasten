Permanent read only storage that will store code executed at boot time.
![[Read Only Memory-20240524202433887.webp|355]]
Most RAM these days are [[Flash ROM]]
# ROM Function
- Stores [[Firmware]]. [[Unified Extensible Firmware Interface|UEFI]] or [[Basic Input Output System|Basic Input Output System]] does this
- Stores the location of the operating system so that the CPU can boot it
- Contains the [[Power-On Self-Test|POST Program]]
The ROM's [[Motherboard Firmware]] only provide a tiny bit of [[Device Driver]]. Additional ones can be installed or given as a CD when purchasing motherboard.
