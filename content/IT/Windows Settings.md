The 'replacement' to [[Windows Control Panel]]. 
# Categories
### Time & Language
Allows for viewing and configuring:
- time
- date
- language
	- You can add keyboards
- region
- cortana speech language
- narrator voice
### Update & Security
- Allows for downloading most recent windows updates
- configuring of [[Windows Security]]
- creating backups and restoring backups
- activating windows
### Personalization
- Background
- Desktop colors
- Folder options
- Icons
### Apps
- Uninstall apps
- Changing autostart toggle
### Privacy
- Limiting telemetry
- Limiting app permissions
### [[Windows System]]
Lots of miscellaneous settings
- Fonts
- Display layout
- Audio input & output devices
- volume levels
- bluetooth
- notification priority
### Devices
Mundane adjustment of devices. Nothing crazy, just like mouse speed changes
### Network & Internet
- Connection settings
- Adapter settings
- Network sharing options
- Dial-up
- [[Virtual Private Network]]
- [[Proxy Server]]
### Gaming
Toggles game mode which prevents windows update from automatically restarting system
### Accounts
Changing user privileges and permissions
