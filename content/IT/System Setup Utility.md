This is a menu accessible at boot time through pressing F2, DELETE, etc. Activation button depends on the manufacturer.
A bios password is required to access this menu.
In this menu, you can:
- Change boot order 
- View hardware information
- Change admin password
- Enable [[Overclocking]]
- Enable wireless
- Enable [[Virtualization]]
- Enable [[Secure Boot]]
- Change [[Universal Serial Bus|USB]] permissions
- Configure [[PC Fan]] settings
- Enable [[Trusted Platform Module|TPM]]
- Load default settings
- Hide hardware from operating system
# BIOS Password
It is advised to always set these passwords to prevent automatic logins
### Administrator Password
Prevents BIOS changes
### User Password
Prevents booting
