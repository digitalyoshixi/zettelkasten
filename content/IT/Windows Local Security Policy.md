Management of security policies on a system.
These policies can be responsible for:
- Setting specific times/curfews for people logged on
- Setting password expiration dates
- Setting password requirements
![[Windows Local Security Policy-20240706214627293.webp]]
To configure group policies across multiple systems, you must use [[Windows Group Policy Management Console]] in [[Windows Active Directory]].
