# Ticket Anatomy
1. User information
2. Date
3. Device information
4. Categories
5. Severity
6. Escalation Levels. Should the problem be given to higher professionals?
7. Problem description (ensure you this is clear and concise)
8. Progress notes
9. Problem resolution (always document your findings)
