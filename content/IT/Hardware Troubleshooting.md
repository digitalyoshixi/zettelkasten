# Troubleshooting Process
1. Make backups
2. Identify Problem
	- Gather information
		- Environment changes
		- Infrastructure changes
3. Test your theory
	- If it is outside your skill level, you need to escalate and ask your supervisor or call the vendor
4. Stick with the client to see if they can perform their task
5. Document findings
	- Problem, Date
	- Key issues
	- Solution, Date

# Overheating
1. Check that PC case is well ventilated
2. Check if heatsink installed right
3. Clean PC
# Grinding/Scraping Noise
Either the fans or the hard drives are in poor quality
# Popping Noise
- Capacitor broke
# Capacitor Swelling
![[Hardware Troubleshooting-20240731193317743.webp|376]]
Certain times capacitors may break. This may cause bumps on the motherboard.
You can replace these by soldering new ones on.
# Very Inaccurate Date/Time
Large time gaps like month - year long time discrepencies are caused by faulty CMOS batteries.
# No Power to Certain Devices
- Faulty [[Power Supply Unit|PSU]]
- Incorrectly oriented [[Serial AT Attachment|SATA]]/[[External SATA|eSATA]] storage cables 
# Slow Performance
- [[Central Processing Unit|CPU]] too weak
- [[Random Access Memory|RAM]] too little
- [[Hard Drive|HDD]] issues
- Too much software running concurrently
