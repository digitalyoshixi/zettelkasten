[[Random Access Memory|RAM]] that uses [[Hamming Codes]] to error correct.
Often has extra pins to detect data corruption with parity checking.
- It wont fix it, it will just detect it.
- ECC RAM sticks may come in 72-bit, rather than 64-bit. The extra 8 bits is used for the ECC.
- ECC is rarely used as there are very few motherboards designed to be compatible with ECC.
