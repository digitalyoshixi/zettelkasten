The [[Complimentary Metal-Oxide Semiconductor Battery|CMOS]] can reset itself to its factory settings.
This is wanted incase problematic changes occur - for example [[Overclocking]] side effects.
# Reseting
### Shunt
1. Unplug computer
2. Move the shunt from wires 1 and 2 to wires 2 and 3
   ![[CMOS Clear-20240525163120590.webp|175]]
3. Wait 10s
4. Move shunt back to wires 1 and 2
5. Boot up system
### Without Shunt
Touch a screwdriver to the jumpers for 10s
   ![[Basic Input Output System-20240817223448396.webp|234]]
### Battery BIOS
Remove the CMOS battery then place it back in
![[Basic Input Output System-20240817223900342.webp|198]]
### BIOS Menu
Reset to default settings in the [[System Setup Utility|BIOS Menu]]
![[Basic Input Output System-20240817223924949.webp|484]]
