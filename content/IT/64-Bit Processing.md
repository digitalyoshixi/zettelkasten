### 64-Bit Processing
The introduction to 64-bit integers allows for a new type of computer, the 64-bit computer.
The primary benefit of 64-bit computing is that it supports more than the 4GB of RAM limit that 32bit processes are bound by
The RAM can support at max $2^{64}$ bytes of memory which is 18,446,744,073,709,551,616 bytes (16 Exabytes), enhancing the performance of most programs.
