# VM Escape
When an attacker or malware can break out of a VM and into the host machine.
Prevent this with:
- [[Sandbox]] (hopefully by a different provider)
- Frequent updates
# VM Sprawl
A large number of virtual machines that are deployed without administrative controls.
These are unmanaged virtual machines.
[[Solarwinds Virtualization Manager]] can remove unmanaged machines.
