A commonly found printer which uses an ink roller to create images.
![[Inkjet Printer-20240725011826880.webp|409]]
Resolution is measured in DPI (Dots per inch)
Print speed is measured in pages per minute (PPM)
# Components
### Print Head
![[Inkjet Printer-20240725012543205.webp|265]]![[Inkjet Printer-20240725014629721.webp|424]]
Head drops small ink drops on the page.
Most printers use heat to move the ink but some do it mechanically aswell.
Some printers combine the ink cartridges and print head into one component. Others separate the two
### Ink Cartridge
![[Inkjet Printer-20240725012403734.webp|484]]
Sold mostly in [[CMYK]] colors
Can also be sold in other colors like dark cyan, green, gray, etc.
### Feed Roller
![[Inkjet Printer-20240725012328191.webp|484]]
Rollers responsible for pulling the paper into the printer.
Must be cleaned often.
# Ink Carriage
![[Inkjet Printer-20240725012841688.webp]]
A rail that allows the print head and ink cartridge to move along the page.
# Additional Features
### Duplexing
Certain printers will print on both sides of the paper without any manual help.
### Calibration
![[Inkjet Printer-20240725014340815.webp]]
In the menu options, you configure the printer cartridges and then a output page is printed out.
Windows uses [[Windows Color System|WCS]] calibrates using [[International Color Consortium|ICC]] color profiles to correct color shifts from OS to hardware.
# Pros and Cons
| Pros                   | Cons                     |
| ---------------------- | ------------------------ |
| Relatively inexpensive | Expensive to replace ink |
| Quiet                  | Ink eventually fades     |
| High resolution        | Clog easily              |
