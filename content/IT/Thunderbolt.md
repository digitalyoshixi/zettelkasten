# Thunderbolt 1
![[Thunderbolt Port-20240515015528778.webp|182]]![[Thunderbolt Port-20240515015540318.webp|185]]
- Data transfer of 10Gbps
- [[Mini DisplayPort]] connection
- Can send [[Peripheral Component Interconnect|PCI]] data
- Can send [[DisplayPort]] data
- Can send [[Ethernet Protocol|Ethernet]] data
- Supports up to 6 devices on a single port
- With copper cables, it can extend up to 3m
- With fiber cables, it can extend to 60m
# Thunderbolt 2
Same as thunderbolt 1 except:
- Data transfer of 20Gbps
# Thunderbolt 3
![[Thunderbolt Port-20240515015828983.webp]]
- [[Universal Serial Bus|USB-C]] design
- Runs at 40 Gbps
- Runs with half the power consumption compared to Thunderbolt 2
- Can send [[Peripheral Component Interconnect|PCI]] data
- Can send [[DisplayPort]] data
- Supports up to 6 devices on a single port
- With copper cables, it can extend up to 3m
- With fiber cables, it can extend to 60m
