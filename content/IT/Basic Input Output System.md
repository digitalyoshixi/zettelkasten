A crappy motherboard firmware.
It is responsible for:
- Booting system
- [[Device Driver|Device Drivers]]
- System configuration
[[Unified Extensible Firmware Interface|UEFI]] has replaced BIOS, but UEFI is also commonly called BIOS.
It has a [[System Setup Utility|BIOS Menu]] for configuration
# BIOS Reset
Used to reset BIOS settings including:
- TPM settings
- Boot order
- BIOS password
Can be done with [[CMOS Clear]]
