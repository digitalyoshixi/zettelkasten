Common type of [[Laptop]] RAM that can be added or removed modularly.
Half the size of [[Dual Inline Memory Module|DIMM]]

On laptops that have 72-pin SO-DIMM, you need 2 sticks of RAM to make a full bank.

# Installation
1. Power off the machine.
2. To find RAM,
	- Lift up the keyboard
	- Open a side panel
3. Pop out existing RAM and pop in new RAM
