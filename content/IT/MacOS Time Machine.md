Tool for periodic backing up and restoring files.
Enables users to:
- Enable/Disable auto backups
- Select backup location
- Filter which files are backed up
- Set which time to backup periodically
# Local or Cloud Storage
Time machine can be setup to be locally stored or cloud stored on [[Apple ICloud]].
If its stored locally, it will start deleting old backups if your disk is full.
# Backup Periods
Time machine will automatically perform:
- Daily
- Weekly
- Monthly
