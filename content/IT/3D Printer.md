Printing 3D creations through additive construction by melting plastic filament or resins on a print bed. 
# Printer Types
### Plastic Printers ([[Fused Deposition Modelling|FDM]])
Filament is melted and cools from air
### Resin Printers ([[Stereolithography|SLA]])
UV light cures the resin so it cools 
# Components
![[3D Printer-20240725032130098.webp]]
### Print Bed
A non-stick bed which allows plastic to rest and cool as the build is created
### Filament or Resin
A spool placed as the material to create the build
### Extruder
Filament dropper.
Can move around the carriage.
### Frame
Holds the printer together. Also mounts the carriage so the extruder can move
