# Controllers
Peripherals must each have a chip designated as the 'controller' to communicate with the [[Central Processing Unit|CPU]].
![[Controllers-20240524193154143.webp]]
The [[External Data Bus|EDB]] connects each of these controllers to the [[Central Processing Unit|CPU]]. 
### List of Controllers
- [[Memory Chip Controller|MCC]]
- [[Keyboard Controller]]
- [[Storage Controller]]
- [[USB Host Controller]]
# Chipsets
![[Controllers-20240524195414403.webp|361]]
Motherboards with independent controllers used to have a lot of chips scattered all over the place.
![[Controllers-20240524195517933.webp|361]]
Motherboard manufacturers started to bundle multiple controllers into chipsets which are a smaller set of chips.
These days, almost all controllers are packed into one chipset.
