# Print Job Never Prints
1. Ensure printer is on
2. Ensure printer is connected
3. Check display for error message
4. Restart printer
5. Open spooler and delete all jobs
6. Right click printer service > *Print directly to the printer*
7. Restart printer service
8. Press OK on the printer
# Strange Size Print
- You configured scaling wrong in page setup
- Recheck printer drivers
- Uninstall or reinstall drivers
- Issue with printer engine
# Grinding Noises
- Too much paper in tray
- Realign paper in paper feeder
- Clean rollers
# Stapling and Hole punch Issues
- Remove stapler cartridge and remove jammed stables, the put cartridge back in
- Compatibility issues
- Paper fed in wrong orientation
# Incorrect Page Orientation
- Wrong setup in page setup
# Garbled Print
- Corrupted driver
- Wrong driver
- Ensure printer language of driver is same as printer language for printer
- Power issue
- Cable issue
# Incorrect Chroma Display
- Print a color calibration page
- Check that app and printer are not in grayscale mode
- Check toner/ink levels
- Ink/toner is dirty or incorrect colors
# No Image on Print Display
- Restart device
- Firmware update
- Take device to service center
# Test Page / Test print
A test print is a print issued directly from the printer itself, usually used when prints cannot be issued remotely.
- If the test print goes out, there is a spooler issue.
- If the test print fails, there is a hardware issue
# Printer Sub-System Not Found
- Printer spooler not started
