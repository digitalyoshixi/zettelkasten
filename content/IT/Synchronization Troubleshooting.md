# Partial Syncs
Caused by network connectivity issues.
- Move the device closer to the router
- Restart the router
# Synchronization Prevented
- [[Operating System|OS]] is blocking synchronization
- Server failing to authenticate client
# Multiple Devices Syncing At Once
[[Race Condition]] or a 1-stream only synchronization configuration may prevent complete synchronizations.
