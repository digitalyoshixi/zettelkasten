Edits operating system and startup processes & services.
# Options
### General
![[Windows System Configuration-20240705190007585.webp|382]]
Can change the type of startup during after boot. whether this is normal or diagnostic (no started services) or inbetween
### Boot
![[Windows System Configuration-20240705190611081.webp|392]]
Allows for advanced boot features. You can:
- See all copies of windows installed
- Set default OS
- Delete OS
- Setup safe boot
- Adjust # of memory & cores used
- Windows OS boot order
### Services
![[Windows System Configuration-20240705191141917.webp]]
Similar to services in [[Windows Task Manager]]. You can enable or disable services running on the PC
### Startup
![[Windows System Configuration-20240705191121330.webp|450]]
Launches [[Windows Task Manager]]
### Tools
![[Windows System Configuration-20240705191200224.webp|448]]
Lists all of the windows tool for system configuration. Handy, but nothing we don't already know.
