# DVI Types
### DVI-D (Digital)
![[Digital Visual Interface-20240730191223797.webp|357]]
Sends only digital signals
### DVI-A (Analog)
![[Digital Visual Interface-20240730191207644.webp|366]]
Sends only analog signals
### DVI-A/D / DVI-I (Interchangeable)
![[Digital Visual Interface-20240730191241646.webp|368]]
Accepts either DVI-D or DVI-A.
Sends both digital and analog signals
