These occur as a result of [[Non-maskable Interrupt|NMI]] interrupts caused by a variety of hardware or software issues.
In mac, it is called the Spinning Pinwheel of Death (SPoD).
# Causes
- Faulty component (RAM, HDD, GPU)
- Hardware incompatibility
- Buggy [[Device Driver]]
- Buggy software
- Bad windows update
- [[Basic Input Output System|BIOS]]/[[Unified Extensible Firmware Interface|UEFI]] issues
- [[Power Supply Unit]] problems
- [[Malware]]
- File system errors
