These are of [[Dual Inline Memory Module|DIMM]] or [[Small Outline Dual In-line Memory Module|SO-DIMM]] variety depending on computer type (laptop or desktop).
![[DDR3-20240730203242077.webp]]
- Not compatible with sockets of any other DDR variety.
- 240-pin [[Dual Inline Memory Module|DIMM]]
- 204-pin [[Small Outline Dual In-line Memory Module|SO-DIMM]]
- [[Memory Profile]] compatible
- Triple-channel or quad-channel compatible. You can have at most 4 sticks in one motherboard
- 8-bit buffer size
- 16GB max
- Lowest speed of 800MHz
