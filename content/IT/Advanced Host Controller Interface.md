A communication standard for the [[Storage Controller]].
Mostly optimized for [[Hard Drive|HDD]], but can also work for [[Solid State Drive|SSD]]
Allows for [[Device Hot Swapping]] and [[Native Command Queuing]].
AHCI mode must be enabled in the [[Basic Input Output System|BIOS]] before you install the operating system.
### Disabling AHCI
You can disable AHCI mode in the [[Basic Input Output System|BIOS]]
