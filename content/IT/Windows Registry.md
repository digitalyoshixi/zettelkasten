A database of critical information in a Windows system.
Stores:
- Hardware information
- Network information
- User preferences
- File types
- Passwords
- Desktop color
Almost everything in windows is configured through the registry, or configures the registry.
# Registry Editors
### CLI
- `reg`
- `revsvr32`
### GUI
- `regedit`
