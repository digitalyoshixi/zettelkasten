# Viewing Shared Folders
![[Windows Computer Management-20240706210259303.webp]]
There exists default administrator shares that cannot be deleted, and reappear at every boot. These are a legacy feature. Don't worry about this
### Hidden Shares
Shares with a `$` at the end are hidden shares. It prevents the share from being displayed in a list of available shares.
However, It does not prevent access if the name is known beforehand
