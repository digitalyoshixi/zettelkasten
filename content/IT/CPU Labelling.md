# Mass Labelling Hysteria
Software vendors named their compatibility labels very badly, and thus caused a lot of confusion
### x86 or x32
Supposed to mark CPU's that use x86 instruction set. But commonly used to mark 32bit software.
Can store $2^{32}$ values
### x64
Used to mark that this application or operating system is compatible with their 64bit CPU
Can store $2^{64}$ values
### x86-64
All x64 processors already use the x86 instruction set, so in essence, this is the exact same label as x64
