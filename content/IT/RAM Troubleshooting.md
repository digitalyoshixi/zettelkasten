# Intermittent Memory Errors
RAM errors can be due to [[Electromagnetic Interference]], Bad software, buggy hardware, etc.
### Symptoms
- Programs taking long to load
- General system sliggishness
- Excessive Hard drive accessing
- Program stall
- System lockups: when your computer freezes
- Page faults include the same address multiple times
# Solutions
- Check power supply
- Replace RAM one at a time
- RAM testing software like [[Windows Memory Diagnostic Tool]] or [[Memtest86+]]
