A real-time monitoring software that logs the performance of your system **overtime**.
![[Windows Performance Monitor-20240705220009458.webp]]
# Tool Types
### Object
System component given characteristics that can be managed by the OS. This includes:
- Processor Information
- Cache
![[Windows Performance Monitor-20240706022219000.webp]]
### Counter
Prefaced with *%*. Tracks specific information about object. These can be:
- Processor time
- Async Copy Read/Write
![[Windows Performance Monitor-20240706022310828.webp]]
### Data Collector Sets
![[Windows Performance Monitor-20240706023618405.webp]]
These are sets of counters that you can load from presets or custom make.
