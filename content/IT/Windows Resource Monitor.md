A more in-depth version of [[Windows Task Manager|taskmgr]]'s resource monitor.
# Tabs
### CPU
Allows starting or suspending processes without killing them
### Memory
Shows a view of the in-use, standby and free memory
### Disk
Shows each process' disk activity
### Network
Shows each process' network activity
