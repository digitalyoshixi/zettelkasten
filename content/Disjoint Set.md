A [[Data Structures and Algorithms|Data Structure]] representing a [[Set]] that can perform set operations.
# Operations
- `make-set(x)`: create a set that contains `x`
- `find-set(x)`: return the set that contains `x`
- `union(S1, S2)`: merge sets `S1` and `S2`
- `union(x1, x2)`: merge sets that  con`S1` and `S2`
