# Operation
For a given vector $v = (v_{1},v_{2},\dots)$
For multi-index $(x_{1},x_{2},\dots)$
$$\frac{\partial}{\partial v} = [\frac{\partial}{\partial v_{1}}]^{x_{1}}[\frac{\partial}{\partial v_{2}}]^{x_{2}}\dots$$
# Example
For vector $v=(x,y,z)$
For multi-index $(1,0,2)$
$\frac{\partial}{\partial v} = \left[ \frac{\partial}{\partial x} \right]^{1}\left[ \frac{\partial}{\partial y} \right]^{0}\left[ \frac{\partial}{\partial z} \right]^{2} = \frac{\partial^{3}}{\partial x \partial z^{2}}$
