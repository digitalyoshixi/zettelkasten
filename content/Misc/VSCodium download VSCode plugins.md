https://stackoverflow.com/questions/74672197/how-to-add-extensions-to-vs-codium-open-source-version-from-github-repos


Find extension.vsix here:
- https://vsixhub.com
Download the .vsix file from the releases homepage
Then in the terminal:
```
codium --install-extension clara-copilot-0.0.1.vsix
```
