A [[Javascript Engine]] used in [[Chromium]] and [[NodeJS]].
![[V8-20251019181142536.webp|492]]
# Structure
1. Take the [[JavaScript]] file, and decode it with a [[Lexer]]
2. Lexer passes to [[Parser]]. Parser generates nodes with certain syntax rules, generates [[Abstract Syntax Tree|AST]]
3. [[Abstract Syntax Tree|AST]] sent to [[Ignition Interpreter]]
4. Each bytecode has operation and [[Bytecode Feedback]]
5. Uses [[Inline Caching]]
6. Uses [[Turbofan Optimizer]] to generate [[Machine Code]]
