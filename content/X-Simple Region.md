# X-Simple Definition
A region $R \subset \mathbb{R}^{2}$ is x-simple if there are continuous functions $\psi_{1}$ and $\psi_{2} : [c,d] \to \mathbb{R}$ s.t:
$$R = \{ (x,y) : \psi_{1}(y) \leq x \leq \psi_{2}(y) \vee c\leq y \leq d \}$$
