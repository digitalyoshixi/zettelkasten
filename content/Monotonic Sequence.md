# Definition
If $\{  a_{n} \}$ is monotonic if its [[Strictly Increasing]] or strictly decreasing
### Formal Definition
1. If $\forall n \in \mathbb{N}, \{ a_{n} \}$ is strictly increasing
2. if $\forall n \in \mathbb{N}, \{ a_{n} \}$ is strictly decreasing
