A block contains: 
- A list of 100 [[Blockchain Transaction|Transactions]]
- A link to previous block
- A [[Nonce]]
- A block hash (computed by [[Consensus Algorithm]])
- A [[Digital Signature]] signed by the party's [[Private Key]]
```
Alice pays Bob $500
Bob pays Charlie $400
```
