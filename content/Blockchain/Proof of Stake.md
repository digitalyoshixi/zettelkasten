A proof for adding a transaction that involves staking a given set of ethereum for a given transaction, to give it credibility.

The stake amount for validation will be distributed to everybody that will verify the stake,
If gas fees are high, transaction fees are [[Gas Burning|Burned]]
# Process
1. Each [[Blockchain Miner|Miner]] will deposit a stake for their chance to be chosen to validate the next [[Transaction Block|Block]]
2. Highest stake miner will validate the block, seeing if all transactions
