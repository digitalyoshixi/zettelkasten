A [[Bytecode Interpreter|Virtual Machine]] capable of running [[Smart Contract|Smart Contracts']] bytecode.
Able to run on most operating systems.
![[Ethereum Virtual Machine-20260517014856267.webp]]
- Stack keeps temporary values for [[Stack-Based Virtual Machine|Stack-Based VM]]
- Memory is a temporary [[Buffer]] for large data structures
- Account storage is a key-value database shared between all accounts
