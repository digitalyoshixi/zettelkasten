A [[Partial Order]] where every pair of elements are comparable.
Satisfies the following properties:
- $a \leq a$ ([[Reflexive]])
- $a \leq b \wedge b \leq a \implies a \leq c$ ([[Transitive]])
- $a \leq b \wedge b \leq a \implies a = b$ ([[Antisymmetric]])
- $a \leq b \vee b \leq a$ ([[Connected Relation]])
