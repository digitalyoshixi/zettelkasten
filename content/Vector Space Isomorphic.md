# Theorem
- If $V$ is a $\mathbb{F}$ [[Vector Space]]
- If $dim(V) = n$
- Then, $V$ is [[Isomorphism|Isomorphic]] to $\mathbb{F}^{n}$
# Idea
- Suppose $\{ \alpha_{1},\dots,\alpha_{n} \}$ is an [[Ordered Basis]] for $V$
- Then, define $T \in L(V,\mathbb{F}^{n})$ by $T\alpha_{i} = e_{i}$
- Where $\{ e_{1},\dots,e_{n} \}$ is the [[Standard Basis]] for $\mathbb{F}^{n}$
