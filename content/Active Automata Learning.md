[[Automata Learning]] from actively probing the system. Does not have a [[Test Corpus]].
# Process
- Continually interact with [[System Under Learning|SUL]] 
- Create a hypothesis for input output behavior seen so far
- Check whether hypothesis models complete behavior of system by [[Conformance Testing]], if not then continue
- If hypothesis matches, add it to our model
