A [[Tree|n-ary Search Tree]] used to represent a [[Hashmap|Dictionary]]/[[Set]].
- Nodes position determines its key
- Each node represents a character in a [[Alphabet]]
- Accepting nodes have special notation $A^{*}$
- Traversing a path will result in a full word accepted in a [[Programming/Language|Language]]
