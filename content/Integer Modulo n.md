A set of [[Integers]] that will be [[Modulus]] by $n$
# $\mathbb{Z}_{n}$
$\{ x | x = a \mod n ,a \in Z \}$
or alternatively, $\mathbb{Z}_{n} = \{ 0,\dots,n-1 \}$
Has two operations:
- $+_{\mod_{n}}$ ($a +_{\mod n} b = (a+b)\mod n$)
- $\times_{\mod n}$ $(a \times_{\mod n} b = (ab) \mod n)$
