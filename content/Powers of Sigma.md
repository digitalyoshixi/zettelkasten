A notation to describe a [[Programming/Language|Language]] in terms of the length of [[String|Strings]] that can be created from a given [[Alphabet]]
# Definition
- Given $\Sigma$ as the [[Alphabet]]
- $\Sigma^{n} = \{ \text{set of all strings length n from } \Sigma \}$
# Example
- $\Sigma = \{ 0,1 \}$
- $\Sigma^{0} = \emptyset$
- $\Sigma^{1} = \{ 0,1 \}$
- $\Sigma^{2} = \{ 00, 01 ,10, 11\}$
- ...
