This is a open interface for a networked device used to communicate to different services.

| Port Type          | Port Range  | Desc                                                                                                                       |
| ------------------ | ----------- | -------------------------------------------------------------------------------------------------------------------------- |
| Well-Known ports   | 0-1023      | Used for system services or applications ([[Hyper Text Transfer Protocol\|HTTP]], [[Simple Mail Transfer Protocol\|SMTP]]) |
| Registered Ports   | 1024-49151  | Registered by user processes or applications ([[Remote Desktop Protocol\|RDP]], [[MySQL]])                                 |
| Dynamic or Private | 49152-65535 | Used for ephemeral or temporary purposes. Randomly assigned to client applications                                         |
