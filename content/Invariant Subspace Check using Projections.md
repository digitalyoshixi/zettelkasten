# Theorem
- Let $T \in \mathcal{L}(V)$
- Suppose $V = W_{1} \oplus \dots \oplus W_{k}$
- Let $E_{1},\dots,E_{k}$ be [[Projection|Projections]] with $range(E_{j}) = W_{j}$
- Then, $W_{j}$ invariant under $T, \forall j \in [1,k]$ $\Longleftrightarrow TE_{j} = E_{j}T, \forall j \in [1,k]$
