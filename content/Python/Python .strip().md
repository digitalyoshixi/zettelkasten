.strip() is a method used to remove whitespaces in front and behind a string. 
"   hello    ". will be stripped down to "hello"

.rstrip() is a method with a similar function, however it will only trim whitespaces to the right.
.lstrip() is similar to rstrip, but it only trims to the left.
