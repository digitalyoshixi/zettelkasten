# Locations
```
/var/log
```
- `auth.log`: stores login attempts, auth failures, security events, [[sudo]] uses
- `syslog`: [[Syslog]], stores system events, kernel, various applications, unexpected crashes, startup activities
- `cron.log`: contains information about creation of [[Cron]] jobs
- `user.log`
- `boot.log`
# Alternative Logging Systems
