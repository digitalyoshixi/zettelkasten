If $\lim_{ x \to x_{0}}f(x)$ exists, then it agrees along all path
# Theorem
If $c(t)$ is a path with $\lim_{ t \to t_{0} }c(t) = x_{0}$
Then, $\lim_{ x \to x_{0} }f(x) = \lim_{ t \to t_{0}}f(c(t))$

Also helpful to show that limits do not exist, if a limit does not agree along any two paths.
