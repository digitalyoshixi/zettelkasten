# Definition
- With $f : \mathbb{R}^{n} \to \mathbb{R}$
- With $g : \mathbb{R}^{n} \to \mathbb{R}$
- If we want to find the local extrema of $f$ subject to $g = c$. ($g = c$ is a [[Level Sets|Level Curve]])
- $f$ is the objective function
- $g$ is the constrained function
