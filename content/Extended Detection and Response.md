A monitoring tool deployed on [[Endpoint]], Network, [[Email Server|Email]] and [[Cloud Computing|Cloud]] environments.
# Features
- Aggregates data from [[Endpoint Detection and Response|EDR]] and [[Network Detection and Response|NDR]]
- Uses [[Machine Learning]] to detect multi-stage attacks
- Automation for workflows after specific events
- Endlessly scalable
