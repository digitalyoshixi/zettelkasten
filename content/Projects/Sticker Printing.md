### Photoshop inches

Ok lets edit the template in photoshop.

Firstly, open the ruler using **CTRL+R**

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_77c28b9f2af02922.png)

right click the top segment where the ruler is and select inches.

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_979cfe60323d041d.png)

ok youre good.

Make sure you have this properties tab open as well.

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_62446148869c9a69.png)

it shows the size of each layer. The template tells us that every layer must have a minimum size of 1inch by 1inch.

We also need each sticker to be spaced out atleast a quarter of an inch apart.

### Sticker outline

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_dbb7ac0607594659.png)

ok so I have pretty much all the designs I wanted. I now need to give them that classic sticker outline. You know, that white background thats like a tenth of an inch long that all stickers have.

This is how its done: [https://www.youtube.com/watch?v=igL2oIGhmCw](https://www.youtube.com/watch?v=igL2oIGhmCw)

right click the layer and select Blending Options

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_9c03e608c6bcbe2a.png)

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_361fd6350cafe14e.png)

go to enable stroke. Then modify the size, position and color accordingly. Usually they are about a 10th of an inch long. So in pixels, that would be 9.6 pixels. We cant have decimal so make it 10 pixels.

  

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_60f67832aec02477.png)

now you may see there is some bleeding into the image. We will have to manually erase that.

Once you have your modifications just rasterize the layer style

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_47c4d48a00c4649.png)

now you can manually erase

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_ec4b255a4e5e4a1b.png)

I did a lot of square selection deletion for this because the shape is pretty much a square with beveled edges.

![](file:///C:/Users/Digit/AppData/Local/Temp/lu17828v2iv72.tmp/lu17828v2ivt8_tmp_26eef8c52aa39c12.png)
