A function $T : V^{n} \to W$ is multilinear if it is linear in each of its **n** inputs
$$T(v_{1},\dots, av_{i} + bv_{i},\dots v_{n}) = aT(v_{1},\dots,v_{n}) + bT(v_{i},\dots,v_{n})$$
# Intuition
A regular linear transform is like 1, whereas a multilinear transform is like n, it takes n inputs rather than one input.
All linear transformations are multilinear transformations.
