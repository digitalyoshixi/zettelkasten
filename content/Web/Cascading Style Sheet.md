# Ids
These are specific rules for specific [[HTML Tags]] or tags with specific ids.
```css
body {
	width : 100%
}
div[type=submit] {
	background-color: #4C4C4C;
}
```
# Classes
Rulesets to apply to any object that has a class
```html
<div class="class1"> </div>
```
```css
.class1{
	border : 2px;
	margin : 9px;
	background-color: #4C4C4C;
	text-align : center;
}
```
