# Themes
All a theme is is just a bunch of [[Jekyll Layout]]s given
### Installation
1. Go to https://rubygems.org/search?query=jekyll-theme and find the package you want
2. Edit `Gemfile` to include:
  
  ```gem
  gem "jekyll-theme-hacker" 
  ```
  3. `bundle install`
  4. edit `config.yml` to have: ```
```
theme : jekyll-theme-hacker
```
