An alternate protocol to [[Hyper Text Transfer Protocol|HTTP]] designed for the [[Indie Web]]/Small web.
`gemini://`
# Protocol
1. Gemini server waits on port 1985
2. Gemini sends a seperate request each time it wants to talk to server
3. [[Transport Layer Security|TLS]] verification with trust-on-first-use
