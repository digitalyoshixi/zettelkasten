# Writing Style .sass
Removes the need for semicolons or curly braces
```scss
nav
	ul
		margin : 0
		padding : 0
		list-style : none
	li
		display : inline-block
```
# Superset Style .scss
```scss
nav{
	ul {
		margin : 0
		padding : 0
		list-style : none
		}
	li {
		display : inline-block
		}
	}
```
