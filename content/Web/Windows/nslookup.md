Tool to view the [[Domain Name]]'s [[IP Address]] equivalent
[[Windows]] version of [[Domain Information Groper]].
Will return the [[IP Address]] when given a hostname
# Command
### `nslookup <hostname>`
![[nslookup-20240717190849929.webp]]
Returns the A and AAAA [[DNS Records]]
### `nslookup -type=txt <hostname>`
![[nslookup-20240730164718179.webp]]
Returns the TXT [[DNS Records]]
