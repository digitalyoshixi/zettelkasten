A legacy feature from [[Windows 7]]
A user who can:
- Install software that does not modify system files or [[Windows Services]]
- Customize [[Windows Control Panel]] resources like printers, date, time and power 
- Create and manage local user and groups
- Access other user's files or folders if they are explicitly given access
- Start and stop services
It is a [[Windows Administrator]] without the danger of modifying critical OS functionality.
