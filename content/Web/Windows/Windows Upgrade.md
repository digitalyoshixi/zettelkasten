# Upgrade Types
### In-Place
Keeping all your files and just changing the OS version.
Done with [[Windows Updates]]
### Clean-Install
Removing all files and installing straight from a bootable drive.
Done with [[Windows Media Creation Tool]]
# Upgrade Restrictions
1. You cannot upgrade to x64 or x86 OSes if your CPU does not support the model
2. Cannot in-place upgrade in [[Windows 8]]. You must install [[Windows 8.1]] first.
