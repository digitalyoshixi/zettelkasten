Rainmeter is desktop ricing through use of skins.

# Skins
Skins are configuration files that change widget looks for things like:
- Time
- Location
- Weather
- Music
![[Rainmeter-20231210232754478.webp|513]]
It is best to install a skin and mix and match.
You can find skins on:
- https://www.deviantart.com/rainmeter/gallery/23941137/skins
- https://www.reddit.com/r/Rainmeter/
# Suites
Suites are groups of skins that complement eachother. These are like giving 'dotfiles' out
