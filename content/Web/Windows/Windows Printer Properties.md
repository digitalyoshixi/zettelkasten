# Tabs
### Sharing
Configures printer sharing like in [[Network and Sharing Center]]
### Ports 
![[Windows Printer Properties-20240823212613024.webp]]
Configure how to connect to this printer
- USB
- TCP/IP port
### Advanced
Allows configuration of:
- Spooler
- Printer time availability
- Separator page
### Color Management
Opens [[Windows Color Management]]
### Security
Configure who has access to printer
### Tray Settings
Configure paper size and default tray
