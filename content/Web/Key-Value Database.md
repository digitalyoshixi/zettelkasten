# Key-Value Database
Data is stored in key-value pairs like [[JSON]] or [[Python Dictionary]].
Data is also stored in the [[Web Server]]'s RAM.
- Cannot store large data or perform queries
- Used as cache for sub millisecond fetches
### Pros
- Caches
- Pub/Sub
- Leaderboards
### Implementations
- [[Redis]]
- [[MemCache]]
