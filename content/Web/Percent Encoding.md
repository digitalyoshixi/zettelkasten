[[Universal Resource Locator|URL]] encoding to encode arbitrary data within a [[Uniform Resource Identifier|URI]] using only US-ASCII characters.
- Special characters replaced with percent sign followed by two hex digits (`%20`)
https://www.geeksforgeeks.org/html/html-url-encoding/
# Syntax
```
scheme://subdomain.domain:port/path/filename
```
# Replacements
- Spaces are replaced with `+` or `%20`
