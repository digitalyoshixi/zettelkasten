# Adding Plugins
1. Find a plugin through github or https://quartz.jzhao.xyz/plugin
2. modify `quartz.config.ts` to add the plugin (example https://github.com/CatCodeMe/catcodeme.github.io/blob/v4/quartz.config.ts)
