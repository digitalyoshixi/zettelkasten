A [[JavaScript]] library that uses client-side JS, to create dynamic webpages.

It is responsible only for making user interfaces.
Maintained by [[Facebook]]
It is:
- [[Declarative]]
- Client-sided. It can be very performance heavy on the end user's device.
- Uses [[Single Page App Model]]
[[React Native]] is for mobile applications.
# React Install
```
npm install react
```
### React Setup
```
npx create-react-app <appname>
cd <appname>
npm start
```
OR
```
npm install create-react-app -g
create-react-app <appname>
```
# React Run
1. `npm install`
2. `npm run dev`
# React Development Concepts
- `index.html` does not change. React is responsible for changing the UI on `index.html` not you.
- `src/App.js` contains the [[ReactJS Component|Components]] for `index.html` to display
- `src/App.css` is generated by `App.js`
- `src/App.test.js` is the test cases generated by `App.js`
- `src/index.js` is the main file that loads `App.js`
