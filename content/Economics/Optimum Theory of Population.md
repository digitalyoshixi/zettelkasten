There is an optimal population level societies should reach.
![[Optimum Theory of Population-20241211202005846.webp]]
This optimal population is reached when output per capita maxes out.
# Formula
$\text{Maladjustment} = \frac{\text{Actual Population} - \text{Optimum Population}}{\text{Optimum Population}}$
- If $\text{Maladjustment}$ is positive, then there is overpopulation
- If $\text{Maladjustment}$ is negative, then there is underpopulation
