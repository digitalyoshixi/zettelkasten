# [[UVP]] Formula
### Function
What the product is intended to do.
- Car is designed to move people
### Features
Additional attributes the product can provide
- Car is AC
### Benefits
Advantages that the consumer sees in this product
- Restaurant is trendy ([[Jerusalem]])
- Frequently intangible (status, emotion, experience)
### [[Unique Value Proposition|UVP]] Value Package
Function + Features + Benefits = Value Package
