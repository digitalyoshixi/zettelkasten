A query language for reading [[Prometheus]] databases.
# Examples
- [[PromQL Get Specific Metric]]
# Commands
- [[PromQL count]]
- [[PromQL rate]]
# Query Types
### Instant Vector
Used for a single query
### Range Vector
Used for a range of values
### Scalar
Used as a simple numeric floating value
### String
Used as a string value
