# Issues
- Lack of standardization. IoT does not have a standardized security framework
- Data privacy. IoT devices collect and transmit vast amounts of data
- Insecure commnication. Oftentimes communication protocols are weak
- Long lifecycles and frequent discontinuation. Manufacturers will stop updating devices
- Physical attacks to IoT can compromise security
- [[Supply Chain Vulnerability|Supply Chain Attack]]
- User awareness. Many users are not aware of IoT risks
# Tips
### Change Default Password
- Since, they are accessible on the internet, it is best to change the default password first
### Put IoT Devices on a VLAN
Do not let IoT devices on the same network as your personal devices.
Put them on a guest network or a isolated VLAN
