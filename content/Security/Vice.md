# Installation
1. `yay -S vice`
# Usage
1. `x64sc` to run the emulator
2. Load the disk into the drive (usually drive 8):
   ![[Vice-20250509165933588.webp]]
3. Load the disk into emulator `LOAD "*",8`:
   ![[Vice-20250509170012153.webp]]
4. Run the disk `RUN`:
   ![[Vice-20250509170038012.webp]]
