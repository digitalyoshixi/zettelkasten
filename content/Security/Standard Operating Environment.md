A set of tested and approved software and hardware with versioning that can operate in the office.
Useful for ensuring all devices are running secure hardware/software.
Most organizations will have a standard operating system image with compliant software.
# Non-Compliant Software
-As an end user, if you get an error with software not being compliant, it is because:
- Software is outdated, update to newer version
- Software is using a personal license (it is hard to verify if its legitimate or pirated)
