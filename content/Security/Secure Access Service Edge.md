A [[Wide Area Network|WAN]] technology that allows security policies defined on the cloud.
Offers end-to-end protection for users regardless of location.
Incorporates [[Identity and Access Management]] and many built-in cloud provided security tools like ([[Intrusion Detection System|IDS]], [[Intrusion Prevention System|IPS]], [[Content Filtering]])
