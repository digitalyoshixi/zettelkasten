A control attack for a [[Peer to Peer]] network in which a large number of pseudo-identities occupy the network.
Involves creating many spoofed nodes.
Can be used to:
- Slow down transactions (dont forward to peers)
Can be used to create [[Transaction Block]] chains that are incredibly large
# Countermeasures
- [[Proof of Work]]
- [[Proof of Stake]]
- [[Proof of Elapsed Time]]
