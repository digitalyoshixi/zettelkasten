The process of removing assets within an organization.
Often we decommission [[Legacy Applications]].
Assumes [[Data Retention]] policy has been followed.
# Decommission Process
1. Ensure you have documentation on all 
2. Sanitize all data within the decommissioned items
3. Wipe data or overwrite ([[Data Destruction]])
4. Burn/shred all papers of things
5. Delete all media related to the item (Hard drives)
