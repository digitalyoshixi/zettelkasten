Magic is well, magic is ambiguous. Magic is something we don’t know! Magic is science, but science is spirituality, but spirituality is magic, and the whole thing gets flipped on its head.

Magic numbers: numbers that we don’t know what the point of it is

An example can be found in the [[Portable Executable]] DOS header, which has 2 ascii bytes 'MZ'. these numbers are signifiers that this is a executable file. MZ refers to the programmer's name Mark Zbikowski
