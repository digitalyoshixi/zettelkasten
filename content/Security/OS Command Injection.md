An attack that involves exploiting another attack ([[Webshell]], etc) to run commands on the server's operating system.
# Useful Commands
### Linux
- `whoami`
- `uname -a`
- `ifconfig`
- `netstat -an`
- `ps -ef`
### Windows
- `whoami`
- `ver`
- `ipconfig /all`
- `netstat -an`
- `tasklist`
