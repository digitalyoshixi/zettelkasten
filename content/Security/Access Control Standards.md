# Standards
- Use authentication protocols:
	- [[Secure Shell Protocol|SSH]]
	- [[Kerberos]]
	- [[Open Auth|OAuth]]
	- [[Security Assertion Markup Language|SAML]]
- Use [[Principle of Least Privilege|PLP]]
- Setup [[Access Control]] (Type is up to you)
- Setup user identity (With [[Smart Card]] or [[Biometric Device|Biometrics]])
- Setup [[Multifactor Authentication|MFA]]
- Setup [[Privileged Access Management|PAM]]
- Setup [[Audit Trail]]
- [[Conditional Access Policy]]
