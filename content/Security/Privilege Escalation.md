An attack that exploits vulnerabilities to gain elevated access privileges.
# Types
### Lateral Escalation
A user account is able to create/get access to another user or their resources.
- [[ADCS Privilege Escalation]]
- [[Insecure Direct Object Reference|IDOR]]
### Vertical Escalation
A user account is given permissions for higher privilege accounts via broken access control.
