Ensuring all input fields complies with a specified [[Regular Expression]] or range.
Can prevent against:
- [[SQL Injection|SQLI]]
- [[Stack Overflow]]
- [[Integer Overflow]]
# Techniques
- [[Indirect Specification]]
- [[Input Escaping]]
- [[Canonical Form]]
- [[Whitelist]] patterns
- [[Blacklist]] patterns (least secure)
