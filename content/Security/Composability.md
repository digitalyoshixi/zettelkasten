The property of two functions with the same properties to hold the same properties if given inputs from the other function.
[[Universal Composability]] attempts to create algorithms that will be secure for all keys and inputs.
# IND-CPA Composability
- It is not obvious that a [[Indistinguishability Under Chosen Plaintext Attack|IND-CPA]] algorithm would remain secure if given the keys meant for another [[Indistinguishability Under Chosen Plaintext Attack|IND-CPA]] algorithm
