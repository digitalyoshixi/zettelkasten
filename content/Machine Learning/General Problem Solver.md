A computer program used to work as a universal problem solver machine that uses a trial and error approach.
Solved problems with [[Means-Ends-Analysis]]
Can solve easy problem like [[Tower of Hanoi]] but struggles with larger problems due to [[Combinatorial Explosion]]
# Algorithm
- Any problem is expressed as [[Well-Formed Formula]] or [[Horn Clause]]
- The clauses can be arranged as a [[Directed Graph]]
- The directed graph can be solved by [[General Problem Solver|GPS]]
