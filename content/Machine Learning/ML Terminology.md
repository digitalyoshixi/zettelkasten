### Feature

[https://toloka.ai/blog/machine-learning-labels-and-features/#features-in-machine-learning](https://toloka.ai/blog/machine-learning-labels-and-features/#features-in-machine-learning)

a feature in machine learning Is an individual measurable characteristic of a object that is being observed.machine learning features describe the characteristics of your training data. In training datasets represented as tables, the features are found in columns.

For example, in image labeling features usually include patterns, colors, and shapes that are present in images, something like fur, feathers, or, a lower-level version such as pixel values.
