A collection of:
- AI Model
- Reasoning loop ([[Chain of Thought]], [[Probabilistic Reasoning Techniques]])
- Tools
It is AI that is able to function on itself without need for human input.
They are:
- Autonomous
- Self-Learning
- Able to interact with the world
- Efficient
- Can collaborate with other agents (See [[Multi Modality|Multi Modal Agents]])
