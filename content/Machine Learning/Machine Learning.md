Machine learning is having data, feeding it into the ML model, and then the machine knows what to do. It can be like feeding the model a bunch of images of cats, a bunch of images of dogs and telling it which are the cats, which are the dogs. Then, if we feed the computer an image it has never seen before, it will recognize immediately that this is either a cat or a dog. There is no human telling the AI specific characteristics of a cat, like a cat has pointy ears and a outstretched tail, the AI just recognizes patterns on its own.

**Machine learning is NOT a machine learning from a human!! That is what a general AI is, but because the computer tells the AI what to do, that is not machine learning**

### Types of ML

There are 3 types of machine learning

1. Supervised learning. The data is classified. You tell it this is an X, this is a Y and you expect it to return the correct solution.
    
2. Unsupervised learning. Provide unclassified data. We just say this is the data, give me some clusters of data that Is close together. We don’t get labels, only clusters. Humans need to manually find patterns.
    
3. Reinforcement learning. We reward models for doing the right actions. Say we have a agent that we want to jump high. The agent can bend its knees, crouch, etc. The agent walks left and right, we give it 0pts. If crouch, minus 2 points because we want you to go up. As soon as the agent goes up a little bit, we give it 0.5 pts. If jump high, then score 2 points. Because of rewards and punishments, the agent learns what to do. It always adapts and trains itself

Neural networks depending on the type used to learn, will think differently. Models are discriminative if they are only based off recognition, all supervised models are discriminative. Models are generative if they imagine data, all unsupervised models are generative.
