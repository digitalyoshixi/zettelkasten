A explainable model version of [[Correctional Offender Management Profiling for Alternative Sanctions]].
Was a very simple algorithm
```
if age is 19-20 and sex is male, then predict arrest
else if age=21-22 and priors = 2-3 then predict arrest
else if priors > 3 then predict arrest
else predict no arrest
```
Had better performance than [[Correctional Offender Management Profiling for Alternative Sanctions]]
