A method used primarily for encapsulation on serial lines, along with [[Point-to-Point Protocol|PPP]]
Originally the [[High-Level Data Link Control Protocol|HDLC]] was designed for the OSI model, thus, in order for Cisco to fit it under TCP/IP model, they had to add a "Type" field that specifies whether this frame is for [[Internet Protocol|IPv4]] or [[IPv6]].
