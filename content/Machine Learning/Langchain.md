A tool to develop [[Large Language Model|LLM]] chains and [[Retrieval Augmented Generation|RAG]] based models
# Installation
```
pip install --quiet --upgrade langchain-text-splitters langchain-community langgraph bs4
```
### Installing Anthropic
```
pip install -qU "langchain[anthropic]"
```
### Installing In-Memory Vector Store
```
pip install -qU langchain-core
```
