# Fibre Optic Network Setup
Fibre optic cables are [[Point-to-point Communication|Half Duplex]] - data can flow either way, but for the most part they are used 1-way.
This means you always require 2 cables for fibre optic networks: One for recieving and one for sending
# Fibre Optic Connector Types
### Straight Tip (ST)
![[Fibre Optic Cable-20240716174901222.webp|317]]
These are [[Point-to-point Communication|Half Duplex]].
Used in [[Single-mode Fibre]]
### Square/Subscriber Connector (SC)
![[Fibre Optic Cable-20240716175102748.webp]]
These are [[Point-to-point Communication|Half Duplex]].
Used in [[Single-mode Fibre]]
### Lucent/Local Connector (LC)
![[Fibre Optic Cable-20240716175039805.webp|318]]
The newest connectors. These are [[Point-to-point Communication|Full Duplex]] and are designed to support 2 fibre optic cables in 1 connector.
Used in [[Multimode Fibre]]
