# Required
- UTP Cable
- RJ-45 Head
# Process
1. Strip the top of the UTP cable jacket off
   ![[Creating Own UTP-20240716205146805.webp|182]]
2. Separate the twisted pairs
   ![[Creating Own UTP-20240716205125485.webp|215]]
3. Use the [[Crimping Tool]] to cut off excess wire so that they are 0.5"
   ![[Creating Own UTP-20240716205324418.webp|237]]
4. Insert the wires according to the [[RJ-45|RJ-45 Connector Pin Standards]]
   ![[Creating Own UTP-20240716205348251.webp]]
5. Insert the RJ-45 Head into the Crimping Press
   ![[Creating Own UTP-20240716205451860.webp|236]]
6. Add a boot to the top
   ![[Creating Own UTP-20240716205656700.webp]]
# Testing
Use a [[Cable Tester]] to test if the UTP can transmit data
