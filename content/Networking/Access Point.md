A device that bridges a wired networks with wireless networks.
Functions identically to a [[Switch]] in the sense that it forwards based off [[Media Access Control Address|MAC Address]].
If you are running multiple access points in the same area, ensure they are running on different [[Frequency Bands|WI-FI Channels]].
