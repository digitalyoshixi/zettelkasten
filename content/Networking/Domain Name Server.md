A server responsible for keeping mappings of [[IP Address|IP Addresses]] to [[Domain Name]] and storing records.
Conventionally seen on port `udp/53`.
# Common DNS Servers
`8.8.8.8` - Google DNS
`8.8.4.4` - Google DNS
`1.1.1.1` - [[Cloudflare]] DNS
`1.0.0.1` - [[Cloudflare]] DNS
`9.9.9.9` - [[Quad9]] DNS
