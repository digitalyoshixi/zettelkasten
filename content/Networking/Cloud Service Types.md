# Specialized Features
### Resource Pools
![[Untitled-1-563354369.jpg]]
A cloud provider can provide use of the same computing resources to multiple users at the same time.
This means several users can use a graphics card, processor or memory at the same time.
Can be a security risk, as the tasks performed on your processor can be viewed by others aswell.
### Elasticity
Cloud provides near infinite scaling of your applications.
You will not have to worry about rebuilding your infrastructure, since the cloud provider will manually scale for you.
### Metered Utilization
You pay for how much you consume and can limit the max usage.
Used to prevent costs from skyrocketing.
### Measured Utilization
Cloud provider simply provides the usage metrics.
### High Availability (HA)
The servers are monitored by automated systems and technicians who ensure that the system will always be running 99.999% of the time.
### File Synchronization
Used in providers like [[Github]], who allow storage of files and file history.
