# Guest Networks
Creating a second isolated [[Service Set Identifier|SSID]] using guest accounts to give access to untrusted users.
It is recommended to disable this.
# MAC Address Whitelist
Only allowing a whitelist of devices that have certain [[Media Access Control Address|MAC Addresses]].
This is not perfect, since [[MAC Address Spoofing]] exists
