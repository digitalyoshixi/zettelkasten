created by [[CERN]]

Asks and solves 3 questions:
- How to name a resource?
	- [[URL]]
- How to requests and serve resources?
	- [[Hyper Text Transfer Protocol|HTTP]]
- How to create [[Hypertext]]?
	- [[Hypertext Markup Language|HTML]]
