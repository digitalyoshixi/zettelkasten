# Theorem
- If $D, D^{*}$ are [[Simple Region|Elementary Region]] in plane
- If $T:D^{*} \to D$ is $C^{1}$ [[Class Functions|Class Function]]
- If $T$ is [[Injective Functions|Injective]]
Then, for any integrable region $f:D \to \mathbb{R}$, we have:
$$\int\int_{D} f(x,y)dxdy = \int\int_{D^{*}} f(x(u,v), y(u,v)) |\frac{\partial(x,y)}{\partial(u,v)}| du dv$$
