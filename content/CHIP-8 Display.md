The display is:
- 64px by 32px tall.
- Each pixel is on or off (boolean value)
- 60Hz screen
- Each sprite consists of 8-bit bytes
- Sprites are 1-15 bytes tall
# Drawing
- Take the bytes of the sprite, and XOR each pixel to wherever you want to draw the sprite
