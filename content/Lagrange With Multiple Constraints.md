# Definition
Given multiple constraints:
- $g_{1} = c_{1}$
- $g_{2} = c_{2}$
- ...
- $g_{k} = c_{k}$
With $\{ \nabla g_{1} (x_{0}), \dots, \nabla g_{k}(x_{0})\}$ being [[Linear Independence|Linearly Independent]]
Let $S$ be the set where all constraints are satisfied with $x_{0} \in S$
Then, $f|S$ has local extrema $x_{0}$ $\nabla f(x_{0}) = \lambda_{1} \nabla g_{1}(x_{0}) + \lambda_{2}\nabla g_{2}(x_{0}) + \dots+ \lambda_{k}\nabla g_{k}(x_{k})$
