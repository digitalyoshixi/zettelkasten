A [[Youtube]] frontend for private video streaming.
# Installation (Manual)
https://docs.invidious.io/installation/#linux
1. `apt install git`
2. `git clone https://github.com/iv-org/youtube-trusted-session-generator.git`
3. `python3 -m venv venv`
4. `source venv/bin/activate`
5. `pip install -r requirements.txt`
