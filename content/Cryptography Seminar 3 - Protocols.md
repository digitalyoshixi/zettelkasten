# Notes
- [[Simulation Security]] used to motivate definition
- [[Simulator]]
- Use machines that contain:
	- Party (innocents)
	- Adversary (Provoke bad behavior)
	- Environment (Detect bad behavior)
- Use a protocol which is an ensemble of machines
- Use a execution - which is how the protocol works (semantics)
- Use a emulation - ability to emulate a fancier protocol using a simple one (make a more secure system with weaker assumptions)
