Arrays inside arrays
## Declaring with Known Length
datatype\[]\[] listname = new datatype \[\#rows]\[\#columns] 

## Declaring with Unknown Adaptive Length
datatype\[]\[] listname = { {row1value1,row1value2}, {row2value1, row2value2} }

## Accessing elements
array\[row]\[column];



For loops are just nested for loops.
