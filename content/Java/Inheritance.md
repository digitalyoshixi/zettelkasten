The idea is we can make a subclass that inherits attributes and methods from a parent class

All objects extend from the base Object class.
Technically all classes we make, are subclasses.

# Family Tree
**Child**: Class that takes from the parent class and extends upon them. The subclass
**Parent**: Class that has a child. The superclass
**Grandparent**: Class that has a child that has a child.
