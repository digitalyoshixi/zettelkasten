# Test
Suppose that $f : U \subset \mathbb{R}^{n} \to \mathbb{R}$ is a class of $C^{2}$ [[Class Functions]]
Suppose $x_{0}$ is a [[Critical Values|Critical Point]]
- $Hf(x_{0})$ is [[Positive Definite Matrix]] $\implies$ $x_{0}$ is a relative minimum
- $Hf(x_{0})$ is [[Positive Definite Matrix|Negative Definite Matrix]] $\implies$ $x_{0}$ is a relative maximum
- $Hf(x_{0})$ is neither $\implies$ no conclusion, [[Saddle Point]]
