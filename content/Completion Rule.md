If a [[Binary Expression]] contains unclassified binary sub-expressions and all ways of classifying them place them in the same class, then it is in that class.
# Example
Even if we don't know how to evaluate some expression, the fact that it is part of a larger expression we can reason that they are comparable and optimize
$$
(\text{there is life elsewhere in the universe}) \vee \top
$$
Then, this is a [[Theorem]]
