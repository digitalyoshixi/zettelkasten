A transport protocol that allows transferring of data over [[Controller Area Network Bus|CAN]].
- [[Point-to-point Communication]]
- Up to $2^{32}-1$-byte payloads
- Flow control management
- No re-transmission
![[ISO 15765-2-20260516032959652.webp]]
Uses parameters:
- RX-ID
  TX-ID
- Extended Addressing
- Padding
![[ISO 15765-2-20260516033123974.webp]]
# PCI Frame Types
- Frame 0: Single Frame
- Frame 1: First Frame
- Frame 2: Consecutive Frame
- Frame 3: Flow control frame
![[ISO 15765-2-20260516033050021.webp]]
