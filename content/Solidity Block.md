A globally accessible data structure that provides metadata and [[Random Oracle|Oracle]] of the blockchain.
```
block
```
# Fields
- `block.timestamp` : [[Unixtime]]
- `block.number` : Current block number
- `block.basefee` : Block's base fee
- `block.gaslimit` : Block's gas limit
- `block.chainid` : Current chain ID
