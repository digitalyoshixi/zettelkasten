A wrapper for [[QEMU]] that makes spinning up virtual machines incredibly easy. Alternative to [[Virt Manager]].
# Installation
```
yay -S quickemu-git
```
# Installing a Disc Image
```
quickget macos big-sur
```
# Running Disc Image
```
quickemu --vm macos-big-sur.conf
```
