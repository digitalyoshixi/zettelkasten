A troubleshooting mode that uses basic [[Video Graphics Array|VGA]] graphics in cases where the graphics driver does not work.
Recommended to run if you get a black screen on boot.
It has a resolution of 640 x 480
# Enabling
![[Windows VGA Mode-20240825174807419.webp]]
After [[Power-On Self-Test|POST Program]], press F8
