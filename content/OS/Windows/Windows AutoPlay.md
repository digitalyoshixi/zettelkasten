# AutoRun
It is a legacy feature in [[Windows Vista]] that automatically runs software from a connected USB drive.
It is advised to disable this
# AutoPlay
A windows feature configurable in [[Windows Settings]] > Bluetooth and Devices > AutoPlay.
It allows you to dictate the behavior of the OS everytime a [[USB Flash Drive|USB Drive]] is plugged in
