### General
![[Windows Internet Options-20240824015732619.webp]]
- Home page
- Tab management
- Browsing history
- Searching
- Appearance
### Security
![[Windows Internet Options-20240824015722998.webp]]
- Trusted sites
- Restricted sites
- Site filtering
### Privacy
![[Windows Internet Options-20240824015716054.webp]]
- Cookies
- Location Tracking
- Pop-ups
- Extension Behavior
### Content
![[Windows Internet Options-20240824015704418.webp]]

### Connections
![[Windows Internet Options-20240824015742188.webp]]
- Configure Internet connection to be broadband or dial-up
- VPN connections
- LAN setting configuration
- Proxy server setup
### Programs
![[Windows Internet Options-20240824015748843.webp]]
- Default editor programs
- Browser extensions
