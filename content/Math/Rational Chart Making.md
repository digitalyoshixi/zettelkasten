Oftentimes a question will ask for intervals of increasing and decreasing intervals.
What it asks for is a chart. In this chart you will have included:
- [[Critical Values]]
The boilerplate chart looks as such:

| Interval        | x < ? | ? < x < ? | ? < x < ? | ... | ? < x |
| --------------- | ----- | --------- | --------- | --- | ----- |
| f(x)            | +/-   | +/-       | +/-       |     | +/-   |
| slope           | +/-   | +/-       | +/-       |     | +/-   |
| change in slope | +/-   | +/-       | +/-       |     | +/-   | 

We don't make x = anything. Because you cannot have a change in slope at one point.

## Example
So we may have a reciprocal function with the equation: 
$y=\frac{2x-5}{x+2}$
We need to find the [[X Intercepts|Critical Points]] and the [[X Intercepts|X Intercept]]
##### X intercept
$0=\frac{2x-5}{x+2}$
$2x-5 = 0$
$x = 5/2$
##### Critical Points
$undef=\frac{2x-5}{x+2}$
$0={x+2}$
$x = -2$

##### Before we Chart, we must graph it
![[Rational Chart Making-20231010223147920.webp|381]]
##### Charting
Our points are 5/2 and -2

| Interval        | x < -2 | -2 < x < 2.5 | x > 2.5 |
| --------------- | ------ | ------------ | ------- |
| f(x)            | +      | -            | +       |
| slope           | +      | +            | +       |
| change in slope | +      | -            | -       |
