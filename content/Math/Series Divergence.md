# Divergence Test
Given $\sum a_{n}$
If $\lim_{ n \to \infty }a_{n} \neq 0$
Then, $\sum a_{n}$ diverges
# [[Proof By Contrapositive]]
1. Suppose $\sum a_{n}$ converges
2. Then, by vanishing condition, $\lim_{ n \to \infty }a_{n} = 0$
3. $\square$
# Example
Check if $\sum_{n=3}^{\infty}n!$ diverges or converges
$\lim_{ n \to \infty }a_{n} = \lim_{ n \to \infty }n! = \infty \neq 0$
Thus, $\sum a_{n}$ diverges
