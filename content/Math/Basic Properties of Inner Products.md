# Properties
1. $\langle x | 0 \rangle = \langle 0 | x \rangle = 0$
2. $\langle x | x \rangle = 0 \Longleftrightarrow x = 0$
3. $\langle x, ay+bx \rangle = \overline a \langle x|y \rangle + \overline b \langle x | y \rangle$
4. $\langle x+y | x + y \rangle = \langle x | x )< + 2 Re( \langle x | y \rangle) + \langle y | y \rangle$
