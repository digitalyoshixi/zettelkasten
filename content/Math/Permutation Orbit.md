An orbit is a set that indicates a complete path from one element in the primary set $S$ to its appearance in $\sigma(S)$.
# Example
With primary set $S = \{ 1,2,3,4,5,6,7,8\}$,
With permuted $\sigma(S) = \{ 3,2,5,6,1,4,8,7 \}$
![[Pasted image 20250819141600.png]]
Then, we get orbits $\{  \{ 1,3,5 \}, \{  2 \}, \{ 4,6 \}, \{ 7,8 \} \}$
From our orbits, we can describe the [[Permutation Notation]] of the set  $\sigma(S) = (1 35)(46)(78)$
