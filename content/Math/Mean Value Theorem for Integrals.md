- Let $a,b \in R$
- Let $a < b$
- If $f$ is cont on $[a,b]$
 Then $\exists c \in (a,b)$ s.t $f(c) = \frac{1}{b-a}\int_{a}^{b} f(x) \, dx$
- Or $\exists c \in (a,b)$  s.t $\frac{F(b) - F(a)}{b-a} = f(c)$
 - Kathleen likes this form: $\exists c \in [a,b]$ s.t $\int_{a}^{b} f(x) \, dx = f(c)(b-a)$
# Alternate Area Theorem
- If $f : R \to \mathbb{R}$ is [[Continuity|Continuous]]
- If $R$ is [[Simple Region]]
- Then, there is a point $(x_{0}, y_{0}) \in R$ s.t 
$$
\int \int_{R} f(x,y)dA = f(x_{0},y_{0})Area(R)
$$
