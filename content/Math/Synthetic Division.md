A better way of division than long division. 
## Rules
1. you can only use this if you are dividing by a linear function. so `(ax + b)`.
2. If there is a leading coefficient `a`, you must divide your divisor by the leading coefficient

1. Set divisor equal to zero. `(ax + b) = 0 ; 3x - 2 = 0 ; x = 2/3`
