A set of [[Upper Triangular Matrices]], can be represented as:
- $W_{1} = span( \{  \alpha _{1} \})$
- $W_{2} = span( \{  \alpha _{1}, \alpha _{2} \})$
- $W_{n} = span( \{  \alpha _{1}, \alpha _{2}, \dots, \alpha_{n} \})$
They have the following properties:
- [[Chain]], as $W_{1} \subset W_{2} \subset \dots \subset W_{n}$
- $T$-[[Invariant]] as $T(W_{j}) \subset W_{i}, \forall 1 \leq j \leq i \leq n$
