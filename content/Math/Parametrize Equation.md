The practice of setting one variable as a function of a parameter $t$
# Example 1
$$x+y = 1$$
We want to parametrize both.
- $y = 1 - x$, replace $t =x$ to get $y = 1-t$
Then, we finally get:
- $x = t$
- $y = 1-t$
# Example 2 : Radius and Height
For the radius of a cylinder $r$ and the height of the cylinder $h$,
We can define a parameter $t$ for the height.
Then, define the radius at a given $t$ height as being
$$r(t)  = \frac{r}{h} * t$$
