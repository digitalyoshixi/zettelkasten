# Definition
A collection of elements where:
- The elements are distinct (distinguishable)
- The order of elements does **not** matter
# Example
Curly braces denote sets
- $S = \{1,2,3,4\}$ is a set of 4 elements $1,2,3,4$
- $A = \{red, blue, green\}$ is a set of 3 elements
- $B = \{ \{1,2  \},\{  3,4\},\{ 5,6 \} \}$ is a set of 3 elements which are all sets
An empty set is denoted as $\emptyset$
