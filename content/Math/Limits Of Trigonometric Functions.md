# Sine
$$\lim_{ x \to \infty } \sin x = DNE$$
$$\lim_{ x \to -\infty } \sin x = DNE$$
$$\lim_{ x \to 0 }\sin x=0 $$
$$\lim_{ x \to 0 }\frac{\sin x}{x} = 1$$
# Cos

$$\lim_{ x \to \infty } \cos x = DNE$$
$$\lim_{ x \to -\infty } \cos x = DNE$$
$$\lim_{ x \to 0 }\frac{1-\cos x}{x} = 0$$
# Tan

	$$\lim_{ x \to -\infty } \arctan x = -\frac{\pi}{2}$$
