| Type | Degree | Standard Form |
| - | - | - | 
| Constant | 0 | f(x) = a | 
| Linear | 1 | f(x) = ax + b |
| Quadratic | 2 | f(x) = ax^2 + bx + c |
| Cubic | 3 | f(x) = ax^3 + bx^2 + cx + d |
| Quartic | 4 | f(x) = ax^4 + bx^3 + cx^2 + dx + e | 

**Representations of each graph will vary. So it would be unwise to draw representations of them**
