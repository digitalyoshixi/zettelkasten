# Relation
The number of sets that comprise a [[Relation]].
With [[Cartesian Product]] $A_{1} \times A_{2} \times \dots \times A_{n}$, with $n$ as the arity.
# Function
The number of arguments for a given function.
- Nilad: 0 arguments
- Monad: 1 argument
- Dyad: 2 argument
