# Process
1. Find the [[Characteristic Polynomial]] for matrix $A$
2. Sub-in the matrix $A$ for all instances of $\lambda$
3. Multiply by $A^{-1}$ for all terms, then isolate $A^{-1}$
4. Solve for $A^{-1}$
