Let $x,y \in R$
Assuming $0 \leq x \leq y$

Then, $|x+y| \leq |x|+|y|$
# Proof
Assume we already proved laws 1-5 for [[Absolute Function]].
Let $x,y \in R$
Consider two cases.
Case 1: Assume $x+y > 0$
Then $|x+y| = +(x+y)$ (by definition of [[Absolute Function]])
We know that $x \leq |x|$ and $y \leq |y|$ so it follows that $x+y \leq |x|+|y|$
Case 2: Assume $x-y < 0$
Then, $|x+y| = -(x+y) = -x-y$ (by piecewise definition of [[Absolute Function]])
We know that $-x \leq |x|$ and $-y \leq |y|$ so it follows that $-x -y \leq|x|+|y|$
Since both cases of the absolute function are fufilled, we can conclude that:
$|x+y|\leq|x|+|y|$
