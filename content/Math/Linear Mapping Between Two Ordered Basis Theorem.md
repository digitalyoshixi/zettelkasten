# Theorem
- For [[Vector Space|Vector Spaces]] $V, W$
- With set $\{ a_{1},\dots,a_{n} \}$ as [[Ordered Basis]] for $V$
- With set $\{ b_{1},\dots,b_{n} \}$ as [[Ordered Basis]] for $W$
- There exists a $T : V \to W$ s.t $T(a_{i}) = b_{i}, \forall i \in [1,n]$
# Intuition
If we know what $T$ does to a basis, then we know $T$ completely
