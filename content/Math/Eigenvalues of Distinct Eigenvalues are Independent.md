# Theorem
- If $\forall i \in [i,n], T(v_{i}) = \lambda_{i}v_{i}$ are distinct eigenvalues an eigenvectors
- Then, the set of vectors $\{ v_{1},\dots,v_{n} \}$ is [[Linear Independence|indep]]
# Intuition
If no two eigenvectors share the same eigenvalue, then we have independence
