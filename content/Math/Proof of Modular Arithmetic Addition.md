# Proof
WTS: $( (a \mod n) + (b \mod n)) \mod n = (a+b) \mod n$

With $a,b \in \mathbb{Z}$, then $a = q_{1}n+r_{1}$, $b=q_{2}n+r_{2}$
### LS
Consider $(a \mod n) + (b \mod n) \mod n = (r_{1}+r_{2})\mod n$
### RS
Consider $(a+b) \mod n = ((q_{1}+q_{2})n + r_{1} +r_{2}) \mod n = (r_{1}+r_{2}) \mod n$
Thus, LS = RS
