# Linear Independence
If there are no constants for a set of vectors that allows any linear combination to sum up to 0.
### Formal Definition
$$a_{1}v_{1}+a_{2}v_{2}+\dots+a_{n}v_{n}=0\implies a_{1}=a_{2}=\dots=a_{n}=0$$
