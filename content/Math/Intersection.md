# Notation
$A \cap B$
$A , B$
# Formal Definition
The formal definition is $A \cap B = \{x | x \in A \wedge x \in B\}$
![[Intersection-20240925133220045.webp]]
# N-th Number of Set Intersection
With sets $S_{1},\dots,S_{n}$
- $S_{1} \cap \dots \cap S_{n} = \{ s | \forall i, s \in S_{i} \}$
