A limit $\lim_{ x \to c }f(x)$ along path $c_{1}(t)$ is equivalent to $\lim_{ n \to c }f(c_{1}(t))$
# Example
Consider limit $\lim_{ (x,y) \to (0,0) } \frac{xy^{2}}{x^{3}+y^{3}}$ along paths $c_{1}(t) = (0,t)$
This will be $f(c_{1}(t)) = f(t,0) = \frac{t^{2} * 0}{t^{3} + 0} = 0$
