A modified version of the already quite heavy [[Behavior Table Method]] for when you have an [[Oblique Asymptotes]]
The only change is that for the infty ranges, we measure it from a [[Behavior Function]] given from the O.A.

Remember that the method we use to get the O.A is long division. And oftentimes, through these long divisions, there is a remainder. This remainder, divided by our divisor is our behavior function.

The new values for the infty ranges will be either ABOVE, if the result of putting infty in the behavior function is positive, and BELOW if putting the infty value results in negative.

ABOVE means its above the **asymptote**.
BELOW means its below **asymptote**.
