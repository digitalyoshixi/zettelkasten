# Definition
A collection of elements $A_{1},\dots,A_{n}$ is mutually independent if for any sub-collection $A_{k}, \dots,A_{j}$
$$P(A_{k} \cap A_{k+1}\cap \dots \cap A_{j}) = P(A_{k})P(A_{k+1}) \dots P(A_{j}) \Longleftrightarrow P(\cap_{i=1}^{m}A_{k_{i}}) = \Pi_{i=1}^{m}P(A_{k_{i}})$$
