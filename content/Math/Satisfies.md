# Definitions
A [[Propositional Formula]] $\mathcal{F}$ satisfies if $\forall \tau^{*}(\cdot), \tau^{*}(\mathcal{F}) = 1$.
- $\tau$ is the [[Entail|Model]]
# Intuition
A formula is valid if every assignment of values to its variables makes the formula true.
In other words, it allows the [[Proposition]] equation to be true.
