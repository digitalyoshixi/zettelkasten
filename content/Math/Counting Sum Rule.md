If a task can be done in:
- $n_{1}$ ways or
- $n_{2}$ ways or
- $n_{m}$ ways
- Where none of the sets $n_{1}, n_{2}, \dots n_{m}$ share the same element,
The there are $n_{1} + n_{2} + \dots n_{m}$ ways to do the task
# With Sets
If there is $E$ and $F$ sets, then the sum of those sets is $|E|+|F|-|E\cap F|$
