A function that forecasts the rate at which an event occurs after waiting a given time $t$, given a [[Probability Density Function|PDF]] and [[Cumulative Distribution Function|CDF]]
# Formula
For a given [[Probability Density Function|PDF]] $f(t)$
For a given [[Cumulative Distribution Function|CDF]] $F(t)$
$$h(t) = \frac{f(t)}{1-F(t)}$$
