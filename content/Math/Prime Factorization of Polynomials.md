# Theorem
Let $f \in \mathbb{F}[x]$ be a non-scalar polynomial, Then, $f = \alpha \cdot p_{1}^{n_{1}} \cdot \dots \cdot p_{k}^{n_{k}}$ where:
- $p_{1},\dots,p_{k} \in \mathbb{F}[x]$ are unique distinct [[Reducible Polynomial|Prime Polynomials]]
- $n_{1},\dots,n_{k}$  are  positive integers
- $\alpha$ is a scalar
# Example Prime Decomposition
### Real Function
1. For $f(x) = x^{3}-1$
2. Then, $f(x) = (x-1)(x^{2}+x+1)$, and all of these factors are [[Reducible Polynomial|Prime Polynomials]]
### [[Complex Number|Complex]] function
1. For $f(x) = x^{3}-1$
2. $x_{1}= \frac{1}{2} + \frac{\sqrt{ 3 }}{2}i$
3. $x_{2}= \frac{1}{2} - \frac{\sqrt{ 3 }}{2}i$
4. $f(x) = (x-1)( x - (-\frac{1}{2} +  \frac{\sqrt{ 3 }}{2}i))(x - (\frac{1}{2} - \frac{\sqrt{ 3 }}{2}i))$
