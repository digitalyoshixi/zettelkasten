A [[Subspace]] that has been shifted by a vector.
Represented as set in the form of $\{ v_{p} + v_{h} :v_{h} \in W \}$ where $W$ is a [[Subspace]].
# Example with $y=mx+b$
This is an affine set represented by $(0,b) + span \{ (1,m) \}$
