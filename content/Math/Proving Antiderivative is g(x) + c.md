If $a<b$
And $f'(x)=g'(x)$ for all $x \in (a,b)$
Then $f(x)=g(x)+c$
# Proof
Apply [[Proving f'(0) has f as constant function]] theorem to $h = f-g$
Then $h'(x)=f'(x)-g'(x) = 0$
Thus $h(x)$ is a constant function

$f = h + g$
As h is a constant, we can denote is as $h = c$
so $f(x) = g(x) +c$
