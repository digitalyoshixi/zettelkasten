This is a [[Integral Domain]] with a divisor algorithm. That is:
# Formal Definition
- With $\mathcal{D}$ as an [[Integral Domain]]
- $\mathcal{D}$ is a euclidean domain if $\forall a,b \in \mathcal{D}$, $\exists$ [[Norm Function|Norm]] on $\mathcal{D}$ $N : \mathcal{D} \to \mathbb{Z}^{+}$ s.t:
	1. $\forall a,b \in \mathbb{R}, b \neq 0, \exists q,r \in \mathcal{D}u$ s.t $a = bq+r$
	2. $a,b \neq 0 \implies N(a) \leq N(ab)$
