When they ask you to check for a valid equation, what you do is you make a chart with left side, right side.

# Example

WIth an equation like: $\frac{12}{x} + x = 8$, if we per say, found a solution that is x = 6, x = 2, what we can do to check if make a chart.

| L.S                | R.S |
| ------------------ | --- |
| $\frac{12}{2}$ + 2 |8     |
| $= 6 + 2$          | 8    |
| $=8$               | 8   |
| **L.S = R.S**      |     |
| $\frac{12}{6}$ + 6 | 8   |
| $= 2 + 6$          | 8   |
| $= 8$              | 8   |
| **L.S = R.S**      |     |
So, this solution is valid.
