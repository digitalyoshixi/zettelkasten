Describing [[Set Theory]] in natural language.
In contrast to [[Axiomatic Set Theory]].
# Definition
- Sets are unordered collections of distinct [[Element|Elements]].
- Sets can be described by listing or by rules
	- $A = \{ 1,2,3 \}$
	- $A = \{  n \in \mathbb{N} | 1 \leq n \leq 3 \}$
# Generality
- [[Sample Space]] $\sim$ [[Universal Set]]
- [[Outcome]] $\sim$ [[Element|Elements]]
- [[Event|Events]] $\sim$ [[Set|Sets]]
# Types
- [[Universal Set]]
- [[Empty Set]]
