# Discrete -> Continuous
You turn an integer into a interval.

| Discrete | Continuous |
| -------- | ---------- |
| x=1      | 0.5<x<1.5  |
| x>2      | x>2.5     |
| x>=2     | x>=1.5    |
| x<3      | x<2.5      |
| x<=3     | x<=3.5     |
