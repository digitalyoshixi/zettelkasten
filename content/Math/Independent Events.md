Is an event that has a fixed probability that doesn't change due to circumstance.
Situations in which the occurrence or non-occurrence of one event has **NO INFLUENCE** on the next event that occurs.

# Definition
$E$ and $F$ are Independent events if and only if $P(F|E) = P(F)$
# Rules
# Product Rule
For [[Compound Events]], the product rule is:
$P(A \cap B) = P(A) * P(B)$
- [[Independent Event Product Rule Proof]]
### Union
P(A or B) = P(A) + P(B) - P(A and B)
