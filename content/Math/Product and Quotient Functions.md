# Product Function
if a function in the form $h(x) = f(x)*g(x)$ its very simply, run x into each function and then multiply the y value gotten from each.

# Quotient Function
$h(x)=\frac{f(x)}{g(x)}$

Domain is the common domain of component function + the restrictions in the denominator
