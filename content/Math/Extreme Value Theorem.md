# Theorem
If $f$ is [[Continuity|Continuous]] on a [[Closed Set|Closed Interval]]
Then, there is an [[Global Extrema]] on that interval
# Alternate Theorem
Suppose $C \subset \mathbb{R}^{n}$ is closed and bounded.
If $f : C \subset \mathbb{R}^{n} \to \mathbb{R}$ is [[Continuity|Continuous]], then there are points $x_{0}, x_{1} \in C$ where $f$ attains its [[Global Extrema|Global Maxima]] and [[Global Extrema|Global Minima]].
# Open Interval No EVT
An open interval does not necessarily have a max/min.
Consider the example where the absolute max/min would be at the endpoints. There would always be a max/min bigger than the existing max/min.
