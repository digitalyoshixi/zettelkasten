# Theorem
- With $T \in L(V,W)$
- If $dim(V) = dim(W) < \infty$
- If $T$ is either [[Injective Functions|Injective]] or [[Surjective Functions|Surjective]], then $T$ is [[Linear Map Inverse|Invertible]]
This is useful since we do not need to prove both injective and surjective.
