For function $f$, the inverse denoted as $f^{-1}$.
These functions only exist if the function is [[Injective Functions]]
# Definition
It is defined by:
$$f^{-1}(y) = x \leftrightarrow f(x) = y$$
# Cancellation Formulas
$$f^{-1}(f(x)) = x$$
For every $x \in A$
$$f(f^{-1}(x)) = x$$
