# Definition
A [[Linear Operator]] $T \in \mathcal{L}(V)$ is self-adjoint/hermitian if 
$$T = T^{*}$$
Or alternatively,
$\langle T(a), b \rangle = \langle a, T(b) \rangle$
# Matrix Definition
$$A \in F^{n \times n}, A = A^{*}$$
