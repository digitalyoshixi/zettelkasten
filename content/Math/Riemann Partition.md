A Riemann partition is a [[Partition]] such that each element is spaced out equally.
# Formal Definition
- $a,b \in R$
- $a < b$
- $f$ is defined s.t $[a,b] \subset dom(f)$
- Let $P=\{ x_{i} \}^n_{i=0}$
	- such that $x_{i} = a + i \triangle x$ where $\triangle x = \frac{b-a}{n}$
For $i=0,1,2,\dots ,n$
