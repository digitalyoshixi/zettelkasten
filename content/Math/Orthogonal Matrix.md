# Definition
A matrix is orthogonal if the [[Matrix Transposition]] is the [[Linear Map Inverse|Inverse Matrix]], then that transposition is orthogonal.
Conventionally, the orthogonal matrix is used for [[Real Number]]
# Formal Definition
A matrix $A \in \mathbb{R}^{n}$ is [[Orthogonal Matrix]] if $A A^{T} = A^{T}A=I_{n}$

(Or $A^{T} = A^{-1} \Longleftrightarrow A \text{ is orthoganal}$)
