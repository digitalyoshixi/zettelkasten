A corrolary from [[Linear Map Lemma]] and [[Linear Independence Extension Lemma]]
# Corrolary
1. For any linearly independent set $\{ v_{1},\dots,v_{m} \} \subset V$ for [[Vector Space]] $V$
2. For any subset $\{ w_{1},\dots,w_{m} \} \subset W$, for [[Vector Space]] $W$
3. There is a [[Linear Transform|Linear Map]] $T \in L(V,W)$ s.t $T(v_{k}) = w_{k}$
