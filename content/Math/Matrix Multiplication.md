# Matrix Multiplication Formula
- $T = [t_{ij}] \in M_{t \times k}(\mathbb{F})$
- $S = [s_{ij}] \in M_{k \times n}(\mathbb{F})$
- Then, $TS = [p_{ij}]$
- Where $p_{ij} = \sum_{n=1}^{k} t_{in}s_{ni}$
# Properties
- Distributive over addition
