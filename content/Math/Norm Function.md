This is a [[Function]] that assigns a positive length to a [[Vectors|Vector]] within a [[Vector Space]] to convert it into a [[Norm]].
# Definition
- $f : D \to \mathbb{R}$ s.t
- $f(x) > 0$
- if $f(x) \neq 0 \wedge f(y) \neq 0 \implies f(x) \leq f(x)f(y)$
