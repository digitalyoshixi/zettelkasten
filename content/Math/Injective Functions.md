1-1 function. Function f maps distinct elements of its domain to distinct elements.

Simply put, all x values will have unique y values.
2 different x values, CANNOT have the same y value.
# Formal Definition
$\forall x_{1},x_{2}\in A:(f(x_{1}) = f(x_{2})) \implies (x_{1}=x_{2})$
### Kernel Definition
$T$ is injective $\Longleftrightarrow$ $ker(T) = \{  0 \}$
