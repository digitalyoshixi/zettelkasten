A set $I$ is comprised of values $a,b \in R$ where $a < b$.

# Operations
- $[a,b] + [c,d] = [a+c,b+d]$
- $-[a,b] =[-b,-a]$
- $[a,b] \times [c,d] = [min(ac,ad,bc,bd), max(ac,ad,bc,bd)]$
- $join([a,b], [c,d]) = [min(a,c), max(b,d)]$
