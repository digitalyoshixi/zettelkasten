A different form of [[Set Notation]]

For inequalities.

If it is >= then write \] 
If it is <= then write \[
If it is > then \(
if it is < then \)

### Conversions
For the interval:
4 <= x < 99
\[ 4, 99)

**its important to note that it must start with a open bracket/square bracket and end with a close bracket/square bracket
This is not possible: (9, x (5 
because that would mean that x is bigger than 9 and bigger than 5 which is redundant**

### Unions
For some inequalities you have 2 inequalities like
{-3 <= x < 7} , 
{x >= 14} 
so u have a number line that is split between 7 and 14.
![[Pasted image 20230906120308.png]]
to do this in Interval Notation, you will need to first convert the individual sets.
\[-3, 7)
\[14,inf) 
then union the 2
\[-3, 7) U \[14,inf) 
### Numberline
