# Theorem
- Suppose $T \in \mathcal{L}(V)$
- let $c_{1} ,\dots, c_{n}$ be distinct [[Eigenvector|Eigenvalue]] for $T$
- With $W_{i} = ker(T - c_{i}I), j = 1,\dots,k$
- Then, the set $\{ W_{1}, \dots, W_{n}\}$ is [[Linear Independence|Linearly Independent]]
- If $T$ is [[Diagonalizable]], then $V = W_{1} \oplus \dots \oplus W_{n}$
