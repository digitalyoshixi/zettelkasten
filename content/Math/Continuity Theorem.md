If a function can be finitely constructed with continuous component functions, then that function is continuous aswell.
![[Continuity Theorem-20241008102910213.webp]]
Remember that if the components are continuous, then you can prove:
- compostions
- additions
- differences
- products
- quotients 
of the functions are continuous aswell
