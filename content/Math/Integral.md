A function that returns the area a function covers from y=0.
An integrable function needs to be continuous, but not [[Differentiability]], 
$f(x) dx$
# Integral Types
### [[Definite Integral]]
- There is a definite range
- You are trying to evaluate an integral
- $\int_{b}^a  \, f(x) dx = F(b)-F(a)$ (by [[Fundamental Theorem of Calculus|FToC]])
- There is no $+\ c$ constant
### [[Indefinite Integral]]
- Does not have a range
- Most integrals are indefinite
- There is always a $+\ c$ constant
