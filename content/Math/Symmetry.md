# Symmetry After Transformation

y = a\[k(x-d)^n\]+c
## Even functions
Vertex: **(d,c)** 
Axis of symmetry: **x=d**

for a > 0, graph opens up. the min is c and range is y >= c
for a < 0, graph opens down and max is c and range is y <= c

## Odd functions
There are no vertex in odd functions
There is only point symmetry in odd functions
