# Example
Verify conclusion of rank nullity for the transformation:
$T_{1} : \mathbb{R}^{3}\to \mathbb{R}^{3}$ for projection to z-axis
![[Drawing 2025-02-14 12.36.20.excalidraw]]
$T_{1}(x,y,z) = (0,0,z)$
$dim(\mathbb{R}^{3}) = 3$
$dim(ker(T_{1})) + dim(Image(T_{1}))$
$= 2 + 1 = 3$

# Example 2
Verify conclusion of rank nullity for the transformation:
$T_{2} : \mathbb{R}^{2}\to \mathbb{R}^{2}$ for rotation
![[Drawing 2025-02-14 12.37.39.excalidraw]]
$T_{2}(x,y,z) = (0,0,z)$
$dim(\mathbb{R}^{2}) = 2$
$dim(ker(T_{2})) + dim(Image(T_{2}))$
$= 0 + 2 = 2$
