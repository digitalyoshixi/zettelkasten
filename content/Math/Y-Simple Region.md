# Y-Simple Definition
A region $R \subset \mathbb{R}^{2}$ is y-simple if there are continuous functions $\phi_{1}$ and $\phi_{2} : [a,b] \to \mathbb{R}$ s.t:
$$R = \{ (x,y) : a \leq x \leq b \vee \phi_{1}(x) \leq y \leq \phi_{2}(x) \}$$
