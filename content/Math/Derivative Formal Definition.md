# Definitions
### [[Difference Quotient|First Principles]]
$$\lim_{ x \to 0 } \frac{f(x+h)-f(x)}{h}$$
### a, b
$$\lim_{ x \to a } \frac{f(x)-f(a)}{x-a}$$
# One-sided Derivatives
Mostly used for [[Piecewise Functions]]
### Left Sided Derivatives
$$\lim_{ h \to 0^- } \frac{f(a+h) - f(a)}{h} $$
### Right Sided
$$\lim_{ h \to 0^+ } \frac{f(a+h) - f(a)}{h} $$
