# Satisfiability Definitions
### Valid Definition
A [[First Order Predicate Formula]] $F$ is valid $\Longleftrightarrow$ every [[Predicate Formula Interpretation]] [[Satisfies]] $F$
### Satisfiable Definition
A [[First Order Predicate Formula]] $F$ is satisfiable $\Longleftrightarrow$ some [[Predicate Formula Interpretation]] [[Satisfies]] $F$
### Unsatisfiable Definition
A [[First Order Predicate Formula]] $F$ is unsatisfiable $\Longleftrightarrow$ all [[Predicate Formula Interpretation]] that [[Falsifies]] $F$
### Falsifiable Definition
A [[First Order Predicate Formula]] $F$ is falsifiable $\Longleftrightarrow$ some [[Predicate Formula Interpretation]] that [[Falsifies]] $F$
