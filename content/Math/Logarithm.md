# Definition
A [[Function]] that returns the exponent you need to get a argument from a given base.
![[Logarithms-20231130155322898.webp|369]]
![[Logarithms-20231129152444178.webp|166]]
- **Base:** $0 < x < 1 ∪ x>1$
- **Argument:** $0<x$
- **Exponential:** $x∈ℝ$
# [[Log Properties]]
Only valid if b is positive
$$\log _b(1)=0$$
$$\log_{b}(b)=1$$
$$\log_{b}(b^x)=x$$
$$b^{\log_{b}x}=x$$
$$e^{\ln x}=x$$
$$log_{b}x = \frac{1}{\log_{x}b}$$

- Logs argument are never negative
- Log bases are never negative unless you use [[Imaginary Number]]
- Log base can never be one
# Theorems
### [[Change of Base Theorem]]
$\frac{\log t}{\log b} = \log_{b}t=x$
### [[Power Law]]
$\log_{b}(x^n)=n\log_{b}x$
### [[Product Law of Logarithms]]
$\log_{b}(mn)=\log_{b}m+\log_{b}n$
### [[Quotient Law of Logarithms]]
$\log _b(\frac{m}{n})=\log_{b}(m)-\log_{b}(n)$
