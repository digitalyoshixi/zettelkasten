# Random Variable
A random variable is:
1. A countable quantity
2. A depiction of an independant trial out of a larger # of trials.
3. Has fixed # of trials
For a random variable X, the variable x are the separate outcomes that X can be.
# Types of Random Variables
#### Discrete
Integers OR
Values that are finished. will never change. Constant
#### Continuous
Decimals OR
Values that are still changing. will always change. dynamic
