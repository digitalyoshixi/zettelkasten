# Univariate
- For a given [[Random Variable|RV]] $X$
- For a given function $Y=g(X)$
- Then, $E(Y) = \sum_{X}^{}g(x)p_{X}(x) = E(g(X))$
# Multivariate
- For given [[Random Variable|RV]] $X,Y$
- For a given function $Z = g(X,Y)$
- $E[Z]=E[g(X,Y)] = \sum_{X}^{} \sum_{Y}^{}g(x,y)P(X=x, Y=y)$
