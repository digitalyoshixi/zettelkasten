# Definition
Let $D = \{ d_{1},\dots,d_{n} \} \subset \mathbb{F}[x]$
Then, the ideal generated by $D$ is the set $F[x]D = \{ f \in F[x] | f= g_{1}d_{1}+\dots+g_{n}d_{n} | g_{i} \in \mathbb{F}[x], \forall i \}$

If $D = \{ 0 \}$, then $\mathbb{F}[x]D$ is the [[Trivial Ideal]]
