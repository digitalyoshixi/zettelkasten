# Definition
- If $D = \{ d \}$ contains only a single element $d \in \mathbb{F}[x]$ 
- Then the [[Finitely Generated Polynomial Ideal|Generated Polynomial Ideal]] $M$ generated by $D$ is called a principle polynomial ideal.
$$M = F[x]D = F[x]d = \{  f \in \mathbb{F}[x]  | f = gd| g \in \mathbb{F}[x]\}$$
