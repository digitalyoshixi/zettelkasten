y = a\[k(x-d)\]^n+c
**a** - Vertical stretch/compress. 
**k** - Horizontal stretch/compress. **1/k is the actual stretch/compress**
**d** - Horizontal shift. **left is positive, right is negative.**
n - Exponent
**c** - Vertical shift

## Vertical Stretch
| Parameter | Effect on graph y=x^n | 
|-|-|
| 1 < a | Vertical stretch by factor of a |
| 0 < a < 1 | Vertical compression by factor of 1/a|
| -1 < a < 0 | Vertical flip on x axis, then compression by factor of 1/a |
| a < -1 | Vertical flip on x axis, then stretch by a |
You can find this from [[Vertical Stretch from Y Intercept]]
## Horizontal Stretch
| Parameter | Effect on graph y=x^n |
|-|-|
| k > 1 | Horizontal compression of k |
| 0 < k < 1 | Horizontal stretch by factor 1/k |
| -1 < k < 0 | Horizontal flip on y axis, then stretch by factor of 1/k |
| k < -1 | Horizontal flip on y axis, then compress of k |

## Horizontal Shift
| Parameter | Effect on the graph y=x^n |
| - | - |
| d > 0 | Horizontal shift left d units |
| d < 0 | Horizontal shift right d units |

## Vertical Shift
| Parameter | Effect on the graph y=x^n |
| - | - |
| c > 0 | Vertical shift up c units |
| c < 0 | Vertical shift down c units |
