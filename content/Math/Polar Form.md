# Polar Form
For a given complex number in the form: $z = a + bi$
$z = re^{i \theta}$
Where:
- $r = |z|$ (See [[Complex Number Modulus]])
- $\cos \theta = \frac{a}{r}$
- $\sin \theta = \frac{b}{r}$
- $\theta$ is considered the argument
![[Polar Form-20250115121353705.webp]]
# Polar Form [[Conjugate]]
$\overline z = \overline{r(\cos \theta + i \sin \theta)} = re^{-i \theta}$
# Specific Polar Forms You Should Remember
- $e^{i2\pi} = 1$
- $e^{i \frac{\pi}{2}} = i$
- $e^{i\pi} = -1$
- $e^{i \frac{3\pi}{2}} = -i$
