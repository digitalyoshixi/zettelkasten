# Theorem
If $T$ is invariant, then:
1. $T$ is upper triangularizable
2. There exists a basis $\{  \alpha_{1}, \dots, \alpha_{n} \}$ for $V$such that if $W_{i} = span(\alpha_{1},\dots,\alpha_{i})$ then:
	- [[Chain]], as $W_{1} \subset W_{2} \subset \dots \subset W_{n}$
	- $T$-[[Invariant]] as $T(W_{j}) \subset W_{i}, \forall 1 \leq j \leq i \leq n$
