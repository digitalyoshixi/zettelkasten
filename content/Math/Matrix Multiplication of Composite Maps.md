# Definition
1. With bases $\alpha = \{ u_{1},\dots,u_{n} \}$
2. With bases $\beta = \{ v_{1},\dots,v_{k} \}$
3. With basis $\gamma = \{ w_{1},\dots,w_{t} \}$
4. If $S : V \to V$
5. If $T : V \to W$
6. Then, the product $[T]_{\beta}^{\gamma} * [S]_{\alpha}^{\gamma} = [TS]_{\alpha}^{\gamma}$
