# Definition
1. Let $a,b \in R$
2. Suppose $a<b$
3. Suppose $f(x)$ is bounded on $[a,b]$
4. $\int_{a}^{b} f(x) \, dx$ exists and is equal if and only if there is a value that both the set of all lower darboux sums and upper darboux sums [[Convergening Series|Converge]] to.
    $\int_{a}^{b} f(x) \, dx = sup \{ L(f,p) | \text{P is any partiton of [a,b]} \} = inf \{ U(f,p) | \text{P is any partiton of [a,b]} \}$
