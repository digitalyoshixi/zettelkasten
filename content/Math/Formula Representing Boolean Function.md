A [[Propositional Formula|Formula]] $F$ can represent a boolean function using [[Propositional Variable]] $x_{1},\dots,x_{n}$ using [[Truth Assignment]].
# Definition
For any [[Truth Assignment]] $\tau$,
- $\tau$ satisfies $F$ if $f(\tau(x_{1}), \dots, \tau(x_{n})) = 1$
- $\tau$ falsifies $F$ if $f(\tau(x_{1}), \dots, \tau(x_{n})) = 0$
