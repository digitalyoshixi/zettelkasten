# Definition
A [[Metric]] is nice $\Longleftrightarrow$ [[Norm]] [[Absolute Value Functions Induce a Metric Theorem|Induces a Metric]]
# Showing a Metric is Nice
1. $\lambda \in \mathbb{F} \implies d(\lambda x, \lambda y) = |\lambda|d(x,y)$
2. Let $z \in V \implies d(x+y, y+z) = d(x,y)$
