# Definition
Disability is defined is having a 'mixed bag' regarding well being.
They are just differently abled.
# Implications
- We can treat [[Autism]] to be different ability rather than disability
- Deaf people don't see it as a disability, some don't want to have hearing ([[Cochelar Question]])
# Responses
