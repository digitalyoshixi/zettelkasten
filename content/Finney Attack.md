An attack that involves:
- Attacker private create chain A which is blocks from $A\to B$ (both attacker controlled)
- Attacker public create chain B which is equivalent number of blocks from $A\to \text{other address}$
- Attacker publishes chain B
- Attacker publishes chain A
- Chain A takes precedence because it was mined earlier
