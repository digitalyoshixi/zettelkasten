A tool used to brainstorm [[Object Oriented Programming|OOP]] software.
# Card Structure
### Class Name
The name of the class.
Also often includes the superclass and subclasses if any.
### Responsibilities
Key responsibilities / themes of the card.
Often start with verbs
### Collaborators
Other classes that this class will work with
