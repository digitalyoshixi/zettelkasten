# Definition
The set of [[Predicate Logic|FO]] formulas.
Comprised of:
- Infinite set of variables $x,y,z,\dots$
- Set of [[Predicate]] symbols $A,B,C,\dots$ with its own [[Arity]]
- A set of constants $a,b,c,\dots$
