# Definition
For a [[Pushdown Automata|PDA]] $M$ to accept a string $x$:
- Start with an empty stack (initial state)
- Read all of $x$
- End in an accepting state
- End with an empty stack
# Example
For $\Sigma = \{ 0,1 \}$
For $L = \{ x \in \Sigma^{*} : \#_{0}(x) = \#_{1}(x) \}$
Find a [[Pushdown Automata|PDA]] that accepts $L$
