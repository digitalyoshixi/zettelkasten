# Process
1. Is the note part of the [[Chord]]? If so, then move it down to the closest chord note
2. Can I harmonize a 3rd or a 6th without sounding too dissonant? Then do that
![[Ricky's Harmonizing Process-20251019032031689.webp|280]]
3. Is the note a perfect 4th above a note in the chord? Then, harmonize it a 4th down
![[Ricky's Harmonizing Process-20251019032331317.webp|262]]
4. None of the above. Then, try and harmonize down to the closest chord note (that is atleast a 3rd away)
5. Still sounds bad. Then make some changes to the melody or chords and try again
