# Definition
For a sequence of [[Independent Random Variable]] $X_{1}, \dots, X_{n}$ with [[Average of Random Variable|Common Mean]] $\mu$, and variance $\sigma^{2}$
the standardized average is:
$$Z_{n} = (\frac{\overline X_{n} - \mu}{\frac{\sigma}{\sqrt{ n }}}) = \sqrt{ n }(\frac{\overline X_{n} - \mu}{\sigma})$$
- Standardized RV will always have $\mu = 0, \sigma=1$
