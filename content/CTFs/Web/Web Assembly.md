Migrating low level software written in [[Rust]], [[C]], [[COBOL]] in a web browser at near native performance.
It consists of web assembly text like [[Assembly]] compiled to WASM binary.
# Creating WASM Binaries
1. Use [[Web Assembly Text]]
2. Use [[Emscripten]] to convert C/C++ program to WASM
3. Use [[AssemblyScript]] to convert [[TypeScript]] into WASM
