The Nextjs version of `<a>` tags.
These allow for better javascript navigation, removing the need for a full browser refresh to load a page.
Nextjs also automatically performs:
- Code splitting
- [[Prefetching]]
# Internet Webpage
```js
<Link href="https://ctftime.org/user/181893">For CTFTime</Link> 
```
# Local Webpage
```js
<Link href="/about"></Link> 
```
