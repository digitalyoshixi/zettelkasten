A [[Server Side Rendering]] js framework.
Based off [[ReactJS]]. 
- Good for webscrapers
- Good for speed
# Setup
```
npx create-next-app <appname>
```
# Project Files
1. `/app` - [[NextJS App Router]]
2. `/pages` - [[NextJS Dynamic Pages|NextJS Page Router]]
3. `/public` - Static assets to be served
4. `/src` - optional source folder
5. `/next.config.js` - nextjs conig
6. `/middleware.ts` - middleware files
7. `/next-env.d.ts` - [[TypeScript]] type declarations
8. [[layout.js]]
