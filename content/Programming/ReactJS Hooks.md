Adds features to [[ReactJS Component|Components]].
Always put them inside components **not** outside of them.
![[ReactJS Hooks-20241101045337104.webp]]
# Types of Hooks
### State Hooks
Managing [[ReactJS State]]
- [[useState()]]
- [[useReducer()]]
### Context Hooks
Lets you use data passed through [[ReactJS Context]]
- [[useContext()]]
### Reference HTML Hooks
Lets you reference things like [[Hypertext Markup Language|HTML]] elements
- [[useRef()]]
### Effect Hooks
Lets connecting with other apps like [[Application Program Interface|API]]
- [[ReactJS Effects|useEffect()]]
### Performance Hooks
Improving app performance by preventing unnecessary work
- [[useMemo()]]
- [[useCallback()]]
