The if-statement continues if the input is **non-zero.**
[[Shorthand If Statements]]
# Single statement conditional
```cpp
if (booleanvalue) //something one;
else // something two
```
# Multiple statement conditional
```cpp
if (booleanvalue){
	// the if condition
}
else if {
	// the other case
}
else {
	// the other case
}
```
