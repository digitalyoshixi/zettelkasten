# Datatypes
- A [[Byte]] is 8-[[Binary|Bit]] `.byte`
- A Half [[Binary|Word]] is 16-[[Binary|Bits]] `.half`
- A [[Binary|Word]] is 32-[[Binary|Bits]] `.word`
- A Integer is 32-bits `.word`
- A non-null-terminated string `.ascii`
- A null-terminated string `.asciiz`
- Reserved byte space `.space`
- aligned $2^{n}$ byte space `.align n` (usually n is $4$ for word alignment)
