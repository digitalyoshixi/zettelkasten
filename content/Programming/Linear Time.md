# O(n)
Linear algorithms.
If the number being provided to us is some number, than this time is linear.
directly proportional lines being ran from a given counter.
## Examples
- [[Linear Search]]
### Example Linear Program
```java
int n = 100;
for (int i = 0; i<n;i++){
	System.out.println("hi");
	System.out.println("hi");
}
```
The above program has function
f(n) = 2n.
the one line of our number equates to 2 lines of code.
