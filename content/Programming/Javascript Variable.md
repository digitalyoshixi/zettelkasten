These store data.
There are 3 ways to create variables:
- var
- let (recommended)
- const
# var
```js
var x = 10;
```
Allows for the highest versaility
# let 
```js
let x = 10;
```
# const
Constants. you cannot update it
```js
const button = document.querySelector("#button1")
```
used for values that will never change.
You are however able to edit that const's attributes.
