A [[Programming/Language|Language]] generated by a [[Context Free Grammar|CFG]].
Set of all context free languages is the set of all languages verified by [[Pushdown Automata]]
# Definition
A language $L$ is context-free $\Longleftrightarrow$ $L = \mathcal{L}(G)$ for some [[Control Flow Graph|CFG]] $G$
