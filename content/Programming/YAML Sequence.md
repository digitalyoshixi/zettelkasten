These are a list of [[YAML Scalar]] or [[YAML Mappings]]
# Sequence of Scalars
```
- Hello
- This
- Is 
- Several
```
# Sequence of Mappings
```
-
  name: Mark McGwire
  hr:   65
  avg:  0.278
-
  name: Sammy Sosa
  hr:   63
  avg:  0.288
```
# Flow Sequence
```
[name        , hr, avg  ]
```
Alternate way to write a sequence.
Also allows for sequences of sequences.
