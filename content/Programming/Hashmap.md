This is a one-way mapping from one datatype to another.
```python
mymap = {
	a : 20,
	b : "hello"	,
	"quint" : 1.2239
}
```
# Operations
- `insert(k,v)`: insert a new key-value pair `k-v`
- `delete(k)`: delete the node with key `k`
- `search(k)`: find node with key `k`
