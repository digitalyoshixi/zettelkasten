Any character that shows up for formatting with no effect on code logic.
This includes:
- ` ` space
- `\t` horizontal tab
- `\n` newline
- `\v` vertical tab
- `\f` form feed
- `\r` carriage return
# One-Line Programs
You can actually write entire programs in just one line, EXCLUDING the `#include` statements
`#include` statements must always be seperated by newlines
```cpp
#include <iostream>
#include <string>
```
