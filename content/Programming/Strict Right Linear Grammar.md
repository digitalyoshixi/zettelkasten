A [[Right-Linear Grammar]] where all [[Production|Production Rules]] map to [[Empty String]] or [[Terminal]] followed by a [[Non-terminal]]
# Definition
A [[Context Free Grammar|CFG]] $G = (V, \Sigma, P, S)$ is strict right linear if:
- Every production $P$ is of the form:
	- $A \to \epsilon$
	- $A \to xB$, $A,B \in V, x \in \Sigma^{*}, |x| \leq 1$
# Theorems
- [[Languages Generated by Strict Right-Linear Grammars are Regular]]
- [[Every Regular Language can be Generated by a Strict Right-Linear Grammar]]
# [[Finite State Automata|FSA]]
A strict right linear grammar represents the structure of a [[Finite State Automata|FSA]].
Given a strict right linear grammar $G = (V,\Sigma, P, S)$, [[Non-Deterministic Finite Automaton|NFSA]] $M = (Q,\Sigma, \delta, s, F)$
- $Q = V$
- For every $A,B \in V$, $a \in \Sigma \cup \{\epsilon\}, \delta(A,a) = \{ B  : A \to aB \text{ is a production in } P \}$
- $s = S$
- $F = \{ A : A \to \epsilon \text{ is a production in } P \}$
