Prints 0 or 1.
0 for false
1 for true
```cpp
bool a {true};
bool b {false};
```
If you want it to print 'true' or 'false'. send `std::boolalpha` to the output stream

# Typecast Int -> Boolean
as long as the integer is >= 1, then the boolean is true,
other case, then its false.
