External data not relevant to the function of a file. Used to provide meta information
# Types
### Descriptive
Essential data about the content of a file:
- Authors
- Keywords
- [[Exchangeable Image File Format]]
### Administrative
Data about management details:
- [[Modified Access Creation|MAC]]
- Access rights
### Structural Metadata
Data about how components are organized:
- Chapters of ebook
- File size
