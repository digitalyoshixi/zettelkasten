The tradeoff between adding new types and adding new functions.
![[Expression Problem-20260228012036495.webp]]
- In [[Object Oriented Programming|OOP]], adding a new type is easy, but adding new functions for each new type is hard
- In [[Functional Programming|FP]], adding a new function is easy, but adding new types that work with each function is hard
