# Attribute Error
Using the wrong methods on the wrong [[Python Datatypes|Python Datatype]].
For example: using [[Python find()]] on a python integer yields an Attribute Error
# Syntax Error
The python syntax is written wrong.
# Type Error
Doing operations between incompatible python types. They are not automatically typecasted.
