The modern implementation of [[List Processor|LISP]] that proritizes many features for practical programming.
# Interpreter Installation
```
sudo pacman -S clisp
```
# Usage

### [[Read-eval-print-loop|Repl]]
```
clisp
```
### Interpret from File
```
clisp myfile.lisp
```
