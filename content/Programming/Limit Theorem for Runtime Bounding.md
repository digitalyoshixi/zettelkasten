Useful for showing that a function is not in a [[Runtime Bounding|Runtime Bound Function Family]]
# Theorem
$$\lim_{ n \to \infty }\frac{f(n)}{g(n)} = \infty \Leftrightarrow f(n) \not \in O(g(n))$$
$$\lim_{ n \to \infty }\frac{f(n)}{g(n)} = \text{finite} \Leftrightarrow f(n)  \in O(g(n))$$
No case for [[Limits Does Not Exist]]
