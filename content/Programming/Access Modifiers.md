# Public
[[Classes]] data is visible to all in the program.
Used for certain [[Method|Methods]] and [[Enums]]
# Protected
[[Classes]]' data is visible to only derived classes
This should be used most of the time.
# Private
[[Classes]]' data is only visible to itself. Would not be visible in derived classes aswell.
