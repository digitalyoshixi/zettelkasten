A concept in [[Object Oriented Programming]] that allows [[Method|Methods]] to have the same name, but take on different [[Function Signature|Function Signatures]]
# Java
You are allowed:
- Change in parameter numbers
- Same number of parameters, different types of parameters in [[Function Signature]]
Not allowed:
- Same parameters, different return types
# Python
You are allowed:
- Change in parameter numbers
You are not allowed:
- Same number of parameters, different types of parameters in [[Function Signature]] (via [[Weakly Typed|Weak Typing]])
- Same parameters, different return types
