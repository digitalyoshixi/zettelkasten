The fastest way to setup [[JavaScript]] frameworks.
- It bundles javascript modules and dependencies
- Serves your code locally during development
Configuration file at `vite.config.js`
# Creation
```node
npm init vite
```
# Run and Host
```javascript
npm run dev -- --host
```
