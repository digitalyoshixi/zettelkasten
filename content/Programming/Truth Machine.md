A quick test to see if a [[Esolang]] is capable of:
- Input
- Output
- Decision
- Repetition 
- Termination
# Truth Machine
- Ask for input.
- If the input is 0:
    - Output 0.
    - Terminate program.
- Whereas if the input is 1:
    - Repeat infinitely:
        - Output 1.
