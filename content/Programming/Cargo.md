The package manager for rust
# Creating Project
### Current Project
```
cargo init
```
### New Project
```
cargo new myproj
cd myproj
```
# Build Projects
```
cargo build
```
# Run Projects
```
cargo run
```
# Check Compilation Readiness
Will check if a program is able to be compiled.
```
cargo check
```
