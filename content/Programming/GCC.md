# Flags
### `-o`
sets the output name
### `-S`
Assembles program and creates `.s` file
### `-g`
adds debugger symbols
### `-wall`
Shows all the warnings
### `-lm`
For linking [[math.h]] library
# `-std=c++20`
For setting the current version to c++ 20
