A hardware-level virtualization management service.
# Installation
1. Get a proxmox ISO and burn it to a [[Universal Serial Bus|USB]]
2. Setup your [[Redundant Array of Independent Disks|RAID]] and [[Domain Name]] during installation
3. After install, run this script: https://community-scripts.github.io/ProxmoxVE/scripts?id=post-pve-install
4. If you want to add another hard drive:
   https://forum.proxmox.com/threads/how-to-add-hard-drive-to-host.119376/
