# Graph Operations
- Adding an edge ($i,j$)
	- Worst Case : $O(1)$
- Remove edge ($i,j$)
	- Worst case: $O(n)$
- Edge query
	- Worst case: $O(n)$
- Adding node
	- Worst case: $O(n)$
- Deleting a node
	- Worst case $O(n^{2})$
