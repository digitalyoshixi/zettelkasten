This is a [[Lazy Evaluation|Lazy Loading]] [[Statically Typed]] [[Functional Programming]] language with [[Type Inference]].
`.hs` [[File Extension]].
# Installation
```
sudo pacman -S ghc cabal-install
```
# Compile & Link
```
ghc -dynamic file.hs
```
# Interactive [[Read-eval-print-loop|Repl]]
```
ghci
```
