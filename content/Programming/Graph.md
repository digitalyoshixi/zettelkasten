A graph is a collection of $V$ nodes linked by $E$ edges to have specific relationships.
![[Drawing 2025-03-12 11.12.59.excalidraw]]
# Formal Definition
Graph has a
- Set of nodes $V$ with a size of $N$
- Set of edges $E$ with a size of $M$
