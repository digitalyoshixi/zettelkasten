A [[Bottom-up Programming]] algorithm for [[Path Finding]]. Uses a [[Stack]]
# Algorithm
1. Base case is when `S = D` or `S = NULL`
2. Keep a list of nodes visited
3. If we are not the base case, then visit all [[Neighbourhood|Neighbors]]
