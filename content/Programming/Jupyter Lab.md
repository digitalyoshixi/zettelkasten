# Installation
[https://jupyter.org/install](https://jupyter.org/install)
1. Setup a [[Python Virtual Environment|venv]]
2. `pip install jupyterlab jupyterlab-vim ipykernel`
3. `ipython kernel install --user --name=myvenv`
4. `jupyter lab` will open a web-console at `http://localhost:8888/lab`
