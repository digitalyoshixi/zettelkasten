# Theorem
The following are equivalent:
- $L = \mathcal{L}(R)$ for some [[Regular Expression|Regex]] $R$
- $L = \mathcal{L}(M)$ for some [[Deterministic Finite Automaton|DFSA]] $M$
- $L = \mathcal{L}(M)$ for some [[Non-Deterministic Finite Automaton|NFSA]] $M$
- $L = \mathcal{L}(G)$ for some [[Right-Linear Grammar]] $G$
- $L = \mathcal{L}(G)$ for some [[Strict Right Linear Grammar]] $G$
