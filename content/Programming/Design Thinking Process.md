A process used for creative problem solving.
![[Design Thinking Process-20251014182800105.webp]]
# Steps
### Emphasize
Understand the target audience
- Do user [[Interview]]
- Create [[Empathy Map]]
### Define
Define the problem statement and create goals
- What is the problem
- Who has the problem
- Where is the problem
- Why does it matter
### Ideate
Create a wall of sticky notes for brainstorming
- Brainstorming
- [[Worst Possible Idea]]
### Prototype
Develop a [[Fidelity|Low Fidelity]] and [[Fidelity|High Fidelity]] prototype.
- Create a vision board
- Rapid prototype [[Paper-Prototype Computer]]
### Test
Run [[Usability Tests]] on the prototypes.
- [[Observational Testing]]
- [[Iterative Testing]]
### Implement
Deploy the working final prototype
