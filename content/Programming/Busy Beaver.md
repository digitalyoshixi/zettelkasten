An unsolvable problem that involves finding the longest running [[Turing Machine|Turing Machines]].
# Problem Description
1. Create a set of all [[Turing Machine|Turing Machines]] that read from a [[Punchcard]] with $N$ rules.
2. The [[Turing Machine]] that runs the longest before [[Turing Machine Halt|Halting]] (and without running in a loop), is denoted as $BB(N) = n$, with $n$ as its number of steps
