The idea of defining a [[YAML Scalar]], [[YAML Sequence]] to another [[YAML Scalar]], [[YAML Sequence]]. [[YAML Mappings]]
Like a [[Hashmap]].
# Mapping Scalar to Scalar
```
file : "/usr/bin/cat"
```
# Flow Style
```
Mark McGwire: {hr: 65, avg: 0.278}
```
Allows for mapping of mappings
# Complex Mapping Keys
```
? - Detroit Tigers
  - Chicago cubs
: - 2001-07-23

```
