A method of using [[Key Fob|Car Key]] to remotely lock and unlock cars.
# Types
- Static Keys (Code is always X)
- Rolling Code (Both devices have same [[Pseudo Random Number Generator|PRNG]] that generates same codes)
# Methods
- [[Amplitude Shift Keying]]
- [[Frequency Shift Keying]]
- [[Phase Shift Keying]]
