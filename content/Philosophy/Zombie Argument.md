**Phenomenal Facts aren't deducible from physical facts**
# Argument
- It is assumed that phenomenal facts supervene on physical facts
- If zombies are possible, then it is possible for there to be two entities, a zombie and a human with the same physical properties, different [[Phenomenology|Phenomenal]] properties
- This implies that [[Phenomenology|Phenomenal]] facts are not supervened by physical facts
- Thus, [[Physicalism]] is false
# Response
- Some believe in the [[Explanatory Gap]]
