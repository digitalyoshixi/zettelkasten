# Non Assisted Dying Argument
```standardform
P1: killing and letting die are morally equivalent
P2: If killing is sometimes wrong, so is letting die
P3: If letting die is sometimes ok, so is killing
IC: Killing (euthenasia) is sometimes ok
C: Since killing is quicker, it is often better
```
