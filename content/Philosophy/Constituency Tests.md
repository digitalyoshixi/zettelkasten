These are tests used to determine if a string of words is a valid [[Phrase]]
# Tests
If one test succeeds, then we can say the phrase is usually a constituent. Ideally, two tests should be used to verify with high degree of accuracy.
- [[Substitution Test]]
- [[Stand-Alone Test]]
- [[Clefting Test]]
