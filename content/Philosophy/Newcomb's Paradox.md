A paradoxical game wherein one player can predict the future with near certainty.
![[Newcomb's Paradox-20250727003126418.webp]]
# Game
1. Two Agents, one reliable predictor and one player
2. Box A is transparent with $1000
3. Box B is opaque, with its contents set by the predictor
	1. If the predictor predicts the player takes both boxes, then it is set with $0
	2. If the predictor predicts the player takes only box B, then it is set with $1,000,000
4. The player does not know the prediction
