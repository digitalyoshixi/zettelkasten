The principle that ensures all patients are given appropriate treatment.
Values [[Fairness]], [[Equity]] and [[Equality]].

In biomedical ethics, this requires compromises to procedures like:
- Not doing [[First Come First Serve]], if a person is in urgent need of care ([[Medical Triage]])
# Principles
1. To each person an equal share
2. To each person according to need
3. To each person according to effort
4. To each person according to contribution
5. To each person according to merit
6. To each persona according to free market exchanges
