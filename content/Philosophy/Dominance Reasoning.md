Reasoning that relies on picking the more dominant option.
# Dominance
A situation $A_1$ dominates $A_2$ if:
- All the outcomes in $A_1$ at least as good as those in $A_2$
# Example
Consider:
- Game $A$ that has $+1$ if win and $-2$ if lose
- Game $B$ that has $+1$ if win and $-1$ if loes
Game $B$ dominates game $A$ since all outcomes in game $B$ are at least as good as those in $A$ and there are cases in which $B$ has better results
