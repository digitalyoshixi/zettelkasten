*Phenomenal properties are identical to physical properties*
Classifying traits and drawing conclusions based off the behavior of an entity.
For [[Physicalism]], seeing that a entity acts as if it has a mind should result in it being conscious (See [[Epistemology Argument By Analogy]])
# Counter-Arguments
### [[Zombie]]
If a entity acts like a human but actually has no mind
### Smart Rock
If a entity has a mind but does not outwardly show that it has one
