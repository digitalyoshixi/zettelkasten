# Desire -> Actions
Actions require desire.
Everytime you act, you act because of your desires.
# Example
If I put a bill into a vending machine, I am doing so because I want a candy bar.

# Versions
### Lesser Version
Desires can come after motivation is set.
Beliefs are the only input for motivation and desire will follow afterwards as a byproduct
### Rigorous Version
Desires and beliefs must both exist as inputs before motivation can be met.
