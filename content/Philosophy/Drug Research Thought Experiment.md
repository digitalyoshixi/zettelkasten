# Thought Experiment
- Experiment with two drugs
- Drug A works moderately well
- Drug B might work better but we dont know
- How can we ethically test drug B?
- To test at all, we must give participants drug B which might harm them
- If we dont test drug B, then we never improve the drug
- If we provide drug B, then we are willingly withholding something that has positive effect to the patient
