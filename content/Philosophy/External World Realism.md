# Anti-Skeptic Argument
```standardform
P1: Perceptual experiences seem to present us with an external world
P2: Best explanation for P1 is that there really is an external world
P3: If hypothesis A provides a better explanation of evidence than hypothesis B, we ought to favour bypothesis A
-- MP (P2, P3) --
C : Therefore, we ought to favour hypothesis that there is an external world
```
