We prove that time-travel is impossible by showing that the inverse-time function does not exist.
1. Note that for the function time $T(x) : V \to W$, $V\to W$ is not one-to-one, as multiple events can lead to the same outcome-event. Then, the map is not [[Injective Functions|Injective]]
2. Note that for the set of all possible events $W$, there are events not possible at all, then the map is not [[Surjective Functions|Surjective]]
3. Then, since it is not surjective or injective, an [[Linear Map Inverse|Inverse Function]] does not exist
Thus, there is no inverse time function.
