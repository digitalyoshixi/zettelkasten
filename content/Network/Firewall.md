A rule-based whitelist or blacklist usually employed on [[Switch|Switches]], [[Router|Routers]], [[Access Point|WAP]], Software for the device itself or as a dedicated [[Server]].
A firewall is able to:
- Use [[Stateful Packet Inspection|SPI]] to create packet rules
- Block ports
- Enable [[Port Forwarding]]
- Block data based off packet inspection
- Be used as a packet encryption endpoint
- Be used as a [[Proxy Server]]
