In a LAN, all devices receive broadcast frames from the main switch.
This frame is equivalent to an [[Ethernet Frame]], but the [[Media Access Control Address|MAC Address]] is always FF-FF-FF-FF-FF-FF. They are used in the following protocols:
- [[Address Resolution Protocol|ARP]]
- [[Dynamic Host Configuration Protocol|DHCP]]
- [[Network Discovery]]
