# Static IP
Most routers by default will be at [[Dynamic Host Configuration Protocol|DHCP]] dynamic ips.
Disabling DHCP blocks unknown addresses from obtaining an address, effectively kicking them from the network.
# Universal Plug and Play (UPnP)
Provides seamless and automatic connectivity of media devices and printers.
Disable this, as you dont want automatic connections.
# Firmware Updates
Only install stable firmware updates.
Bad firmware updates will brick the router.
# Credential Changing
Change the default:
- SSID
- Password
# [[Media Access Control Address|MAC Address]] + [[IP Address]] Filtering
Only allow packets from known IP addresses and MAC addresses.
