Old protocol similar to [[Internet Message Access Protocol|IMAP]] for handling incoming e-mail. Stores e-mails locally.
Conventionally seen on port `tcp/110`.
Secure port is `tcp/995` ([[Secure Post Office Protocol 3|Secure POP3]])

[[Apple]] does not endorse POP3
# Alternatives
- [[Simple Mail Transfer Protocol|SMTP]]
- [[Simple Mail Transfer Protocol Secure|SMTPS]]
- [[Secure Post Office Protocol 3|Secure POP3]]
- [[Internet Message Access Protocol|IMAP4]]
- [[Secure Internet Message Access Protocol|Secure IMAP4]]
