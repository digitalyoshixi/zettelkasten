Used to divide networks or to make networks that are 'virtually' connected.
# Dividing Networks
You can break up a physical [[Local Area Network|LAN]] into multiple VLANs.
Each VLAN functions the exact same as a regular LAN. Those within the same VLAN can only communicate with eachother.
# Virtual Network
Used to group computer hosts that may not be physically on the same network
