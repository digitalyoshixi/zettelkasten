Cables where both ends share the same [[RJ-45|RJ-45 Cable Standard]].
Used for both ends in [[Patch Panel]] installation
# Use Cases
Used for connecting [[Media Device Interface|MDI]] devices to [[Media Device Interface Crossover|MDI-X]] devices. Like:
- Workstation <-> [[Switch]]
- [[Router]] <-> [[Switch]]
