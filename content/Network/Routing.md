Ability for a packet to be sent to its destination through use of routing tables.
Handled by [[Router|Routers]].
# Routing Algorithms
### Fixed / Static
Administrator manually adds routing information.
It must be manually modified at every network change
##### Fixed Central
- A central server determines the routing information
##### Distributed
- Each node has their own routing information
### Dynamic
Routing information is dynamically updated after every network change.
