Used during [[TCP & IP Data Layer|Data Layer]] for the receiving device to check if data arrived correctly. 
It is an [[n-Error Detecting Code]] similar to [[Cyclic Redundancy Check|CRC]] but solely used on [[High-Level Data Link Control Protocol|HDLC]] and [[Point-to-Point Protocol|PPP]] encapsulation methods.
