# Network Changes
Retrace your changes whether it be:
- An encryption key change
- A power outage, sag or surge
# Cant Access Certain Resources
If the user can access internet but cant access shared files, it is probably a permissions issue
# Hardware Troubleshooting
Open [[Windows Device Manager]] and look for any errors or conflicts with the wireless adapter.
# Firmware Updates
[[Access Point|WAPs]] will often have monthly firmware updates to fix issues in connectivity and performance.
# Signal Improvements
1. Change [[Access Point|WAP]] placement
2. Upgrade [[Dipole Antenna|Dipole Antennas]]
3. Change the frequency channel to avoid interference
4. Remove metal objects that may create dead spots
# Intermittent Connectivity
- Someone else is using the same frequency
- Low signal strength
- Signal is bouncing off objects
- See wireless interference with [[Signal to Noise Ratio|SNR]]
