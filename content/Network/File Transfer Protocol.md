This is a method of file sharing and provides file management functionality like:
- Listing files
- Adding files
- Deleting files
- Renaming files
`tcp/20` handles data transfer
`tcp/21` handles control
# FTP Servers
These servers require authentication with username and password or authentication with a generic/anonymous account.
