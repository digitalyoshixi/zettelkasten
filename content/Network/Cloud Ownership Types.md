# Public Cloud
Outsourcing your cloud needs to [[Amazon Web Services|AWS]] or [[Microsoft Azure]].
Your data is stored on their servers and you don't own any infrastructure.
These are refered to as multi-entity/multi-teanant
# Private Cloud
Creating a personal cloud for your own business to use.
This is the more expensive option, and you effectively become your own cloud provider
# Community Cloud
A private cloud used by several organizations.
Owned by the community that uses it.
Often used by military contractors, who work with other contractors
# Hybrid Cloud
Combining public, private and community clouds.
Only storing sensitive data on private and community clouds and offloading the more computer intensive tasks on the public cloud.
