# Protocol Types
### Connection-Oriented
- [[Transmission Control Protocol|TCP]] based
- No data is lost during connection
- A formal connection setup and close
##### Examples
- [[Hyper Text Transfer Protocol Secure|HTTPS]]
- [[Secure Shell Protocol|SSH]]
### Connection-less
- [[User Datagram Protocol|UDP]] based
- Some data can be lost during connection without much of an issue
- No protocol to establish connection
##### Examples
- [[Dynamic Host Configuration Protocol|DHCP]]
- [[Trivial File Transfer Protocol|TFTP]]
