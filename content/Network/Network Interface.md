A interface for a device to communicate with the computer network.
- [[Network Interface Controller|NIC]] always have a [[Media Access Control Address|MAC Address]]
- [[Virtual Network Interface|VNI]] may only have a  [[Media Access Control Address|MAC Address]] if it is doing [[OSI Data Layer|OSI Layer 2]] communications
