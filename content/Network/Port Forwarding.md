A technology that serves to:
- Map local addresses to ports on the router's global address
- Create tunnels that treat one port's output and the input of another port
# Port -> LAN Address Mapping
A response to [[Network Address Translation|NAT]], where all local addresses are mapped to a single global address.
![[Port Forwarding-20240726031609096.webp|532]]
Port forwarding allows you to map a port on the global address to a local address in the [[Local Area Network|LAN]]. 
# Port Triggering
![[Port Forwarding-20240726032109655.webp|434]]
A trigger port forwards data to a destination port.
