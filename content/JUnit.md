This is a testing library for [[Java]].
You can run tests with [[Maven]], by using `mvn test`
# Dependency with [[Maven]]
```xml
<dependency>
    <groupId>junit</groupId>
    <artifactId>junit</artifactId>
    <version>4.13.2</version>
    <scope>test</scope>
</dependency>
```
# Methods
### .assertEquals()
```java
assertEquals(2,2); // ok!
```
