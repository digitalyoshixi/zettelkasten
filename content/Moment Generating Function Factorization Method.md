# Method
For given [[Moment Generating Function|MGF]] of [[Random Variable|RV]] $X$,
The [[Random Variable|RV]] $Y,Z$ where $X = Y+Z$ can be found by factoring the [[Moment Generating Function|MGF]]
# Example
$X$ with [[Moment Generating Function|MGF]] $e^{(\lambda_{1}+\lambda_{2})(e^{t}-1)}$
Then, $Y$ with [[Moment Generating Function|MGF]] $e^{(\lambda_{1})(e^{t}-1)}$
Then, $Z$ with [[Moment Generating Function|MGF]] $e^{(\lambda_{2})(e^{t}-1)}$
