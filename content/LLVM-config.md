A tool used to find configurations for LLVM projects.
```c
llvm-config --cxxflags
```
# Example [[Clang]] Compiler flags
```
-I/usr/include -std=c++17   -fno-exceptions -funwind-tables -D_GNU_SOURCE -DEXPERIMENTAL_KEY_INSTRUCTIONS -D__STDC_CONSTANT_MACROS -D__STDC_FORMAT_MACROS -D__STDC_LIMIT_MACROS
```
