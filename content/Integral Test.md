# Theorem
- Given $\sum_{n=1}^{\infty}a_{n}$
- If $f(x)$ is positive
- if $f(x)$ is [[Continuity|Continuous]], on $[1, \infty)$
- If $f(x)$ is [[Strictly Decreasing]] on $[1,\infty)$ 
- if $\forall n \in \mathbb{N}, a_{n} = f(n)$
- Then, $\sum_{n=1}^{\infty}a_{n} \text{ converges } \Longleftrightarrow \int_{1}^{\infty}f(x)dx \text{ converges }$
# Intuition
Both converge or both diverge
