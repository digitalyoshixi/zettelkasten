# Definition
1. If $T$ is diagonalizable, we can compute the projection $E_{i}$ using the [[Lagrange Polynomial]].
2. For $1 \leq j \leq k$
3. Then, $p(x) := \Pi_{i \neq j} \frac{(x - c_{i})}{c_{j} - c_{i}} = \frac{(x-c_{1}) \dots (x-c_{j-1})(c-c_{j+1}) \dots (x-c_{k})}{(c_{j}-c_{1}) \dots (c_{j}-c_{j-1})(c_{j}-c_{j+1}) \dots (c_{j}-c_{k})}$
