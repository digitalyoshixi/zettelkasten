- File has `.sh` file extension 
- All bash scripts must start with the [[shebang]]: `#!/bin/sh` to run the bash shell
- scripts are ran like: 
	- `sh thefile.sh`
	- `./thefile.sh` if the sh file has `chmod a+x` permissions
