A config file for [[Neovim]] that includes:
- plugins
- themes
- dictionary
# Installation
```
git clone https://github.com/NvChad/starter ~/.config/nvim && nvim
```
# Change Theme
Themes are located here: https://nvchad.com/themes/
Change themes in normal mode with `SPACE + T + H`
