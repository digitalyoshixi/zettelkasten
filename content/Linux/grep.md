Finding occurances of a string within a given input.
# `grep <pattern> <input>`

# `grep -rl "string" <directory>`
Recursively search through all files for matches of a specific string.
# `grep -i "string"
Search for a string and dont care about case
# `grep -B3 -A3 "string"
Search for string and also include 3 lines before, 3 lines after
