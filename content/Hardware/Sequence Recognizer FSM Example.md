# Problem 
Create a [[Finite State Automata|FSM]] of a circuit that detects 3 [[Voltage At the Collector|Logic High]] in a row assuming single input $IN$ and two states
# Process
1. Create a state diagram
2. Define transitions dependent on $IN$
3. Create a [[State Table]]
4. Assign [[Flip-Flop]]
5. Create the second [[State Table]]
6. Create a [[Karnaugh Map|K-Map]] for circuit design
