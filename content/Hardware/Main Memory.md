# Structure
Like register files, memory is made up of a decoder and rows of memory units.

# Memory Array
![[Main Memory-20250624144800498.webp]]
This is a structure comprised of several single-cell storage like:
- [[SRAM Cell]]
- [[DRAM Cell]]
Used together to create main memory units.
It has wires deemed as:
- Wordline (used to address words)
- Bitline (used to address bit)
And accepts signals:
- Read
- Write
