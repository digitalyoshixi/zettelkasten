A range of [[Gate Arrays]] that allows [[Hardware Description Languages]] to be [[Code Synthesis|Synthesized]] to change the function of circuits.
Achievable with:
- [[Lookup Tables]]
- [[Routing Fabric]]
Oftentimes, the FPGA is comprised of:
- [[Input Output Blocks]]
- [[Logic Blocks]]
- Routing channels
- Config which you can attach to a [[NAND Chip]] for non-volatile configurations
