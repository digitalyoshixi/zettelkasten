# Definition
- A function $f$ belongs to class $C^{1}$ if all its [[Partial Derivative]] exist and is continuous everywhere on its domain
- A function $f$ belongs to class $C^{k}$ if all its partial derivatives are $C^{k-1}$ on its domain.
